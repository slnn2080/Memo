## あげる　もらう　くれる　等相关用法
用在物品互换 互赠的时候使用

<br>

### あげる: 给
我 为对方做了某件事情, 翻译并不一定要用给, 只是方便记忆

あげる, 主要表示我给对方提供了一些事物 或者 信息等 类似这样有方向性的或者说供给性质的动词还有
1. あげる 给
2. 貸す 借给
3. 教える 教谁

<br>

**要点:**  
1. 主语是我, 我是提供者
2. 人物に, 这里的人物指的是 接收者

```s
友達にプレゼントをあげた。

友達に本を貸した。

子どもに英語を教えています
```

<br>

### もらう: 收到
我 得到了某些事物 或者是 信息, 翻译并不一定要用收到, 只是方便记忆

类似这样有方向性的或者说供给性质的动词还有
1. もらう 收到
2. 借りる 借进
3. 習う　 学习

<br>

**要点:**  
1. 主语是我, 我是接收者
2. 人物に, 这里的人物指的是 **提供者**
```s
山田さんにはなをもらいました # 收到了 山田 的花

香里奈さんにCDを借りました # 从 他那里 借来了 CD

わんさんに中国語を習いました # 我 跟 王 学习中文
```

3. 当提供者不是个人而是机构的时候 我们要使用 から
```s
銀行からお金を借りました # 我跟银行贷了款
```

<br>

### くれる
上面的 あげる 不能用于, 别人给我 或者 别人给己方家属等情况, 当我们想表达别人给己方的时候 我们就要使用 くれる
```s
佐藤さんは私にはなをくれました
```

<br>

### 总结:
上面的3个句子是用在物品互换 互赠的时候使用, 而下面加て形的三种用法, 首先是表达动作方向, 同时表达受惠或使对方受惠

比如下面的句子, **本质上就是 本を貸す, 翻译是也正常翻译** 但是加上了 てあげしまた 就表示 受惠或使对方受惠 的情况
```s
私は木村さんに本を貸してあげしまた
```

<br>

### て形 あげる
**强调说话人为他人提供某种帮助或好处。**, 什么样的好处是根据动词来的

当出现**动词てあげる**时, 表达了说话人做某事为别人带来利益或好处的意愿或行动。这强调的是说话人主动为他人提供帮助或好处。
```s
手紙を書いてあげる。（我给他写信。）
```

<br>

### て形 もらう
**强调说话人从他人那里接受帮助或好处。**, 什么样的好处是根据动词来的

当出现**动词てもらう**时, 这表达了说话人从别人那里接受帮助或好处。这强调的是说话人作为接受方, 从他人那获得了帮助或好处。
```s
友達に宿題を手伝ってもらった。（朋友帮助我做了作业。）
```


### て形 くれる
**也是强调从他人那里接受帮助或好处**, 但更强调施惠者的好意和行为, 常用于描述亲近人之间的行为。

当出现**动词てくれる**时, 这也是表达说话人从别人那里接受帮助或好处, 但与もらう相比, くれる更强调施惠者的行为和好意, 通常用于亲近的人之间。
```s
友達が私に本を貸してくれた。（朋友借书给我了。）
```

## て形 相关句型

### 动词て形 ください
用于指示, 委托, 劝诱对方做什么的时候

<br>

### 动词て形 います
1. 动作进行时
2. 表示状态

```s
私はカメラを持ってます　# 我有相机, 这里持ってます有两种意思 1. 拿在手里, 2. 拥有 的意思
```
3. 习惯性的行为, 职业等
```s
スーパーでナンプラーを売ってます
```

### まだ 动词て形 いません
表示说话的时候 事态还没有发生 或 行为还没有结束
```s
銀行はまだ開いていません
```

<br>

### 动词て形 もいいですか

<br>

### 动词て形 いけません

### 动词て形 动词て形 动词
**1. 列举动作或状态**  
当你想要列举两个或更多的动作或状态时, 可以使用动词的て形。这类似于英语中的"and", 用于连接句子或短语

```s
図書館で本を読んで、勉強して、友達に会いました。(我在图书馆读书、学习、并见了朋友。)
```

<br>

**2. 动作的顺序**  
表达一系列动作按照一定顺序发生时, 可以使用动词的て形来连接这些动作, 表明它们是依次发生的

```s
朝起きて、顔を洗って、朝ご飯を食べました。(早上起来, 洗脸, 然后吃早饭。)
```

<br>

### 动词てから、动词
翻译为先怎么样, 然后怎么样, 前面的行为为前提或者是准备动作

<br><br>

## 动词て形: 表示原因
前面表示原因和理由, 后半部分表示前面的原因带来的结果

因为 ... 所以

<br><br>

## 他动词てあります
表示为了某种目的, 进行的某一种动作, 其结果或状态仍然存在

用来表示某个动作已经被完成, 并且结果持续到现在, 说明某个状态是由过去的某个行为造成的, 并且这个状态或结果至今仍然有效或存在。  

接他动词之后, 表示某人行为动作的结果留下的某种状态。根据上下文的关系, 有时也可带有"为将来做好某事"的意思。

**有点事先预先的意思, 我做这件事情的时候就是为了某件事准备的**

<br>

**使用场景:**  
1. 描述物品的制作或准备状态
2. 表明某个安排或决定已经被做出

<br>

**区别:**  
这个结构强调的是动作的完成和其结果的持续性。它与"ています"不同
- "ています"更多用于表示动作的进行状态或习惯性动作。
- "てあります"主要用于他动词, 强调某个动作被某人完成, 而这个动作的结果持续到现在。

<br><br>

## 动词ておきます (事先做好, 事先完成了)
为做某种准备而有意识地进行的动作。翻译为 **事先** 或者 **之前**

根据语境有时可以表示这是一种临时的措施, 有时也可以表示这是为将来做的准备。

```s
旅行の前に、切符を買っておきます
```

1. 表示在某一时间之前结束必要的动作 行为
2. 表示以备下次使用而完成必要的动作或采取临时的措施
3. 表示将成为结果的状态一直保持下去

<br><br>

## と思う

### と思う
翻译为 我想 我认为 我觉得

主语一般是自己, 但在疑问句中为被翻译为听话人, 你认为怎么样?

<br>

### と思っている
表示在说话之前就已经有这种想法了, 而且现在依然是这样认为的, 含有"早就想好, 计划好, 以前就有这种打算"的意思。

主语是自己 也可以是第三人称

```s
李さんは日本語が難しいと思っています。 # 第三人称
```

<br>

### と思った
表示过去的观点或想法

主语是自己 也可以是第三人称

<br>

### 示例
1. 表达个人意见或感想
```s
明日は雨が降ると思います。(我认为明天会下雨。)
```

2. 表达决定或打算
```s
もう帰ると思います。(我想我要回去了。)
```

3. 推测或猜想
```s
彼はもう睡んでいると思う。(我猜他已经睡了。)

そのニュースは本当だと思いますか？(你认为那条新闻是真的吗？)
```

<br>  

### と思われる  
一般认为 **一般都这样觉得** 用于表达自己的客观判断  

<br>

### 意向型と思う
这里也是表达 说话人想要做什么的意思

**意向型と思う 和 意向型と思ってます:**  
这两个句型都是用做说话人想要做什么, 只不过后者表示这一想法已经存在一段时间了, 同时后者还可以表达第三人称的想法

<br><br>

用于表示说话人的不确定性、疑问或推断, 相当于英语中的"I wonder..."或"Maybe..."。

<br>

1. 表达轻微的疑问或好奇
2. 自言自语中的推测

```s
もう少し安かったらいいのになあ。（要是再便宜一点就好了。）
```

## 被动型

### 直接被动句
1. 主语是动作行为的直接接受者, 动作的行为 **直接** 涉及 **接受者本身**
```s
# 语法
主语は／が、动作实施者に　他动词＋(ら)れる。

王さんは　犬に　噛まれた。(小王被狗咬了。)
```

2. 主语是动作行为的直接接受者, 行为 **直接** 涉及 **接受者的某部分或所有物**
```s
# 语法
主语は／が、动作实施者に　所有物を　他动词＋(ら)れる

王さんは　犬に　かばんを　噛まれた
```

<br>

### 间接被动句
动作行为 **不直接** 涉及被动句的主语, 但是给主语带来了麻烦或损害

```s
# 语法: 自动词
主语は／が、动作实施者に　自动词＋(ら)れる。

せっかくの休みなのに、雨に降られて、どこへも行けなかった


# 语法: 他动词
主语は／が、动作实施者に　动作对象を　他动词＋(ら)れる。

私はレストランで周りの人にタバコを吸われると、気分が悪くなる。
```

<br>

### 自然表明句
客观地描述某种时态和状况, 动作主体为某个不确定的人或群体

有的时候**不是特别注重行为者本身**, 而把事情或者物品作为主题 这时使用被动时态

```s
主语(动作对象)は／が、 他动词＋(ら)れる。  

2021年のオリンピックは、東京で開かれました。
```

<br>

### 补充: 动作施动者后用除了「に」, 还有以下助词的使用
|谓语动词|助词选择|
|:--|:--|
|传递意义动词 送る、届ける、渡す、与える、伝える　など|から|
|创作、发明、建造等意义的动词 作る、書く、発明する、建てる、設計する　など|によって|

## 使役被动形
当某人**被迫或不由自主地**做某件事情时使用使役被动。

### 变形:
其实使役被动是动词
- 先变成使役态后
- 再变成被动的形态。

<br>

1. 1类动词: い段 -> あ段　＋　されます
  - い -> わ: 買います -> 買わされます
  - し -> さ: 話します -> 話させられます
```s
飲みます　飲まされます
```

2. 2类动词: ます -> させられます

## 副助词は & 格助词が

### は: 强调的是后面
除了上面接在ではあります, 强调否定之外 它可以表示一句话的主题

强调的是 は后面的部分 **は(强调部分)**

### が: 强调的是前面
强调的是 が前面的部分 **(强调部分)が**

```js
// 提问的是谓语的部分
    ↓
你是谁？

// 在谓语这里是要使用 は 来进行提示, は后面的学生是我们要了解的新信息

// 提供的部分在前面 也就是提问的部分在谓语的外面
↓
谁是学生？

// 所以要使用 が, 强调的是 が 前面的 私是我们了解的新信息

```

同样是 我是学生 这句话 强调的部分不同 我们会分别使用 副助词は 和 格助词 が

## さえ的使用方式  
さえ 是副助词 相当于提示初次 はも 等用于加强语气的作用  

<br>

**名词さえ...ない**    
```s

# 连学生都没有来。

```

<br>

**助词さえ**    
也是加强语气, 形成助词重叠, 相当于　では、でも、には等  


```s
# 只要不是雨, 就好了。

# 只要是给你做的, 再难也没有关系。
あなたにさえ　作れば　辛くてでも　かまいません。

# 只要不下雨, 明天的运动会按计划进行。 
雨さえ　降らなければ　明日の運動会は予定どおり行われます。
```
 
这里的"さえ"是代替了格助词が、也是为了加强条件句子的语气。

## と的用法  

**1. 表示同一动作的共同参与者**     
和...一起的意思  

<br> 

**2. 表示并列**    
AとB  

<br> 

**3. 表示做交互的对象**    
和A吃饭了  

<br> 

**4. 表引用, 表示所说的内容**    
附在句子（简体句）后面, 简体句＋助词「と」。当直接引用某人的话语时, 通常用「」（日本引号）括起来。当间接引用的时候, 注意句子一定是简体句。后常接续表示说话、思考的谓语动词, 比如：「言う」「思う」「考える」等等。  

```js  
昨日、先生が「明日試験するよ。」と言いました。  
    // 昨天, 老师说："明天要考试。"  

漢字の書き方に注意しなさいと先生が言いました。   
    // 老师说要注意汉字的写法  
```  

<br> 

**5. 表示思考判断的内容**    
「と」表示引用的时候, 加上「思う」、表示说话人（通常为第一人称）思考、判断的内容。附在日语简体句后面, 简体句＋と思う。  

这个用法是上一个用法的扩展, 在「と」表示引用的时候, 加上「思う」、表示说话人（通常为第一人称）思考、判断的内容, 相当于中文的"我认为…""我觉得…"。如果表达第三人称思考、判断的内容, 一般使用「…と思っている」的形式  

<br> 

**6. 表示假定条件**    
动词基本型＋「と」  相当于中文的"假如…""如果…"。  

表示自然且必然成立的条件。在我们判断句子的时候, 需要判断前项和后项是否为必然。
