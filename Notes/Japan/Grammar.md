# 语法:  

### 谓语:
谓语是一个句子中说明主语动作、状态或特征的部分。

它通常**包括动词**，用来**描述主语在句子中的行为或状态**。在一个简单的句子中，谓语可以是一个动词

例如：“他跑了”中的“跑了”，这个动词说明了主语“他”的动作。

在更复杂的句子中，谓语可能由多个词组成，包括动词、助动词、情态动词等，但它们都用来说明主语的动作、状态或特征。

<br>

### 定语:
定语是指**修饰名词或代词的成分**，用来限定或描述这些名词或代词的性质、特征或状态。

在一个句子中，定语通常位于被修饰的名词或代词之前或之后，用来对其进行修饰或说明。

定语可以是
- 形容词
- 名词
- 代词
- 数词
- 介词短语
- 从句 等

它们的作用是**为名词或代词添加更多的信息**，使句子更加具体和丰富。

例如，在句子“蓝色的天空”中，“蓝色的”就是一个定语，它修饰了名词“天空”，描述了天空的颜色。

<br>

### 介词:
介词是一种用来表示名词、代词、动词或其他句子成分之间关系的词语。

它通常位于名词、代词或动词之前，用来说明它们与其他成分之间的位置、方向、时间、关系等。

介词可以用来表示地点、方向、时间、原因、目的等各种关系。

例如，在句子“在桌子上”中，“在”就是一个介词，表示物体“桌子”与某一位置之间的关系，即“在”桌子的“上面”。常见的英文介词包括“in”、“on”、“at”、“from”、“to”等。

<br>

### 代词:
代词是一种用来代替名词或名词短语的词类。

在汉语中，代词可以指人、事物、地点、时间等，通常用来避免重复使用同一个名词，从而使句子更加简洁和流畅。代词根据其指代的对象和用法可以分为以下几类：

<br>

**人称代词:**  
指代说话人、听话人或其他人

主格：我、你、他、她、它、我们、你们、他们、她们、它们

宾格：我、你、他、她、它、我们、你们、他们、她们、它们（汉语中主格和宾格形式相同）

<br>

**指示代词:**  
用来指示具体的人、事物、地点或时间

近指：这、这里、这儿、现在

远指：那、那里、那儿、那时

<br>

**疑问代词:**  
用来提出问题，询问人、事物、地点、时间、原因、方式等

<br>

### 副词:
副词是一种用来**修饰** 动词、形容词、其他副词或整个句子的词类。

它们主要用于提供关于时间、地点、方式、程度、频率等的信息，以进一步描述动作或状态。副词在句子中起着补充说明的作用，使表达更加具体和生动。根据其功能和用途，副词可以分为以下几类

<br>

**时间副词:**  
表示动作或状态发生的时间

例如：今天、现在、马上、已经、刚刚、曾经、从前、以后

<br>

**地点副词:**  
表示动作或状态发生的地点

例如：这里、那里、上面、下面、外面、里面、前面、后面、附近

<br>

**方式副词:**  
表示动作或状态的方式或方法

例如：慢慢地、快速地、小心地、认真地、高兴地、快乐地

<br>

**程度副词:**  
表示动作、形容词或其他副词的程度

例如：很、非常、特别、极、十分、稍微、稍许、几乎

<br>

### 从句:
从句是一个句子中的一个分句，它包含一个主语和一个谓语，并且不能独立成句，必须依附于另一个主句而存在。

从句通常用来进一步说明主句中的内容，起到修饰、说明、补充等作用。

从句可以是名词性从句、形容词性从句或副词性从句，具体根据从句在句子中的作用而定。

**名词性从句：**  
在句子中充当名词的作用，可以用作主语、宾语、表语等。

例如：“他说他明天会来”中的“他明天会来”是一个名词性从句，充当了主语。

<br>

**形容词性从句：**  
在句子中充当形容词的作用，修饰名词或代词。

例如：“我喜欢的书是你给我的”中的“你给我的”是一个形容词性从句，修饰了名词“书”。

<br>

**副词性从句：**  
在句子中充当副词的作用，用来修饰动词、形容词、副词等。

例如：“他在吃饭的时候，经常看电视”中的“在吃饭的时候”是一个副词性从句，修饰了动词“看”。

<br>

### 及物动词:
及物动词是指需要接受一个宾语才能构成完整的意思的动词。

换句话说，这类动词的动作**必须作用于一个对象**，否则句子就会缺乏完整性。例如，"吃"、"看"、"打"、"写"等动词都是及物动词，因为它们需要一个对象来接受动作的影响。

<br>

### 不及物动词:
不及物动词是指不需要宾语就能构成完整意义的动词。

这些动词表达的动作或状态不需要作用于一个特定的对象。例如："跑"、"笑"、"睡"等。

<br>

### (直接)宾语:
宾语是**及物动词所作用的对象**，是动作的接受者或者影响者。

在一个句子中，及物动词通常需要一个宾语来完整表达动作的对象。

例如，在句子 "我吃苹果" 中，"苹果" 就是动词 "吃" 的宾语，因为它是动作的对象，是被吃的东西。

<br>

### 间接宾语:
间接宾语是指**动作的对象，但不是直接受动作影响的人或物**

在汉语中，通常用介词"给"或"对"来表示间接宾语，而在英语中，则通常用介词"to"或"for"。

例如，在句子"我给她一本书"中，"她"就是间接宾语，因为动作的对象是书，而书直接影响的是"她"。

```s
在句子"我给她一本书"中，动词"给"表示的是一种传递或者转移，而书是被传递的对象，因此是直接宾语，而"她"是接受这一动作的人，因此是间接宾语。
```

<br>

### 双宾语动词:
双宾语动词是指能够接受两个宾语的动词，一个是直接宾语，另一个是间接宾语。

<br>

### 副词:
副词基本功能是修饰 动词 或 形容词

<br>

### 用言 (日语):
日语中拥有词尾变化的词 叫做用言, 它们是可以独立做谓语的
- 动词
- 形容词

<br>

### 体言 (日语):
它包含
- 名词
- 数量词
- 代词

它是实词 没有任何词性变化的, 这些词也是没有办法结句做谓语的

## 动词分类

### 根据动词变形的规律 分为
1. 1段动词
2. 5段动词
3. か变 和 さ变

<br>

### 根据动词是否需要宾语 分为
1. 自动词: 如果我们的聚焦点在于状态, 那就用 自动词, 门开了

2. 他动词: 如果我们的聚焦点在于我打开的门, 那就用 他动词 我把门打开了

<br>

### 根据动作是否可持续 分为
1. 瞬间性动词
2. 持续性动词

<br>

### 根据 词性 分为
1. 动作性 动词
  - 跑 跳 打 闹 听 写 睡 学习

2. 状态性 动词 (分为两类)
  - 一类: 单纯的描述存在的状态 いる　ある, 它们就是表示现在有的一个状态, 它们没有ている的形式
    - お金がある: 不能理解我将来有钱, いる　ある 就是表达现在的一个状态, 我们就翻译为现在有钱, 跟ます形不一样 原型/ます形 可以表示反复习惯性的动作, 将来的动作, 而いる　ある 就是表达现在的一个状态

  - 二类: 形容词含义的状态动词 这种动词我们在做谓语的时候 大多数使用 ている
    - 込む - 込んでる　表达一种混杂的状态 这里就表达结果状态的持续

## 助词:

<br>

### に・で・へ・から・まで・と　＋　は
上面的一系列助词后面都可以接 は, 通常情况下表示 对比 如
```s
# 用我的房间 和 其它方法 进行对比, 表示我的房间是没有电话的 但是其他的房间可能有电话
私の部屋には電話がありません

# 韩国 和 中国进行对比 表示韩国去了 但是中国没有去
韓国へは行きました、中国へは行きませんでした
```

<br>

### と
1. N と N: 表示 和
2. N + と + V: 表示 和谁共同做某事

<br>

### へ
表示移动行为的目的地 或者 方向, 前面接场所 后面接移动行为的动词, 如
- 行きます
- 来ます
- 帰ります

<br>

### の
表示　所属关系
```s
わたしの鍵
```

2. 表示　内容
```s
日本語の本
```

<br>

### に
**时间点 に 动词:**  
动作进行的时间, 在带有数字的 **具体时间名词** 的后面要加上 に
```s
１０月に行きます
```

我们在看具体时间的名词的时候, 我们要看是绝对时间点 还是相对时间点
- 绝对时间点 后面 接 に (如: 7点)
- 相对时间点 后面 不接 に (如: 上周, 比如 明日就不能接 に)

<br>

**N(场所)に  动词:**  
动作结束时的位置(场所)
```s
ここに座ります
バスに乗ります
```

<br>

**N(场所)に います:**  
表示存在的场所

<br>

**N(人)に 动词: 表示间接宾语**  
表示 动作行为的作用的对象(方向 或 接受者), に表示动作的接受者

```s
母に　誕生日のプレゼントを　送りました

友達に　メールを　送ります

友達に　プレゼントをあげます

りさんに　手紙を　書きます

母に　電話を　かけます

友達に　お金を　借ります

友達に　お金を　貸します
```

<br>

**N(附着点)に 动词:**  
表示 人 或 物体 的附着点, 动作的着落点

附着点的理解, 比如
- 坐在椅子上
- 乘公交车

乘坐交通工具 或者 **人的身体 或 某物体 停留在某个场所 或者 停留在某个地点**

```s
いすに　座ります
バズに　乗ります
```

<br>

将某个东西塞进什么东西里面 也就使用 ``に`` 来表示
```s
この部屋に　入ってはいけませんか

書類を　カバンに　入れます
```

<br>

**N(目的地) に 动词:**  
移动行为的目的地

在表示移动行为的目的地这一点上, 助词 へ 和 に 没有什么区别

- へ: 它更倾向于表示 方向
- に: 它更倾向于 目的地 的 点

```s
李さんは　病院に　行きます
```

<br>

**时间 に 量词:**  
```s
一週間に　三冊読みます
```  

这里的 に 不要理解为 频率, 而是 に 代表平均分配, 一周来三本, ``ひとりに一個``, 平均分一个人一个苹果

<br>

### で
**交通工具で、行きます、きます、帰ります (移动行为的动词):**  
表示使用什么交通工具去哪里

<br>

**(工具, 手段, 原材料)で + V动词:**  
使用什么完成某件事情

```s
手紙を　速達で　送りました　(手段)

新聞紙で　紙飛行機を　作りました　(原材料)

何で　うどんを作りますか
こむぎこでうどんを作ります
```

<br>

**N场所 + で + V动词:**  
表示 动作进行的场所

<br>

**N场所 + で + 事件 が　あります:**  
这里的 あります 表示 **发生, 举行** 的意思

```s
# 如果 が 的前面 放的是一件事情的话, あります则表示 发生, 举行
ここで　友達の結婚式が　あります

昨夜、大阪で地震が　あります
```

<br>

**N(数量) + で:**  
在不称重量 以个的形式销售物品时, 表达总计的意思
```s
５枚で　２５０円です
```

当数量是1个时, 不加 で
```s
このケーキは　１個　２００円です
```

<br>

### 何 + で的时候 何的发音
**规则1:**  
何 位于辅音 t d n (た、だ、な行假名)前 读 なん, 位于其它辅音前 读 なに

<br>

**规则2:**  
当询问的性质或者种类的时候 读 なに, 询问数量的时候 读 なん

比如 何人 它的发音有两种
1. なにじん: 询问的是哪国人 哪里人, 回答是中国人
2. なんにん: 询问的是数量, 有多少人

<br>

**规则3:**   
**以下情况的发音为 なに**
- 何＋と
- 何＋が
- 何＋を
- 何＋で (交通工具)

<br>

**以下情况的发音为 なん**
- 何＋の
- 何＋で (为什么)
- 何＋时刻/星期

<br>

# 疑问词

### どんな + N(名词)
询问 人 或 事物的 **性质**, 翻译: 什么样的

该疑问词为连体词, 后面需要加N词

当我们回答该疑问词的提问时, 使用 名词短语 (にぎやかな町)

<br>

### 何の
该疑问词询问的是内容 和 材料, 我们回答的时候需要回答内容

```s
何の　本ですか
自動車の本です

何の料理ですか
魚の料理です

どんな料理ですか
からい料理です
```

<br>

### どうですか (でしたか)
询问对方的感想, 翻译: 如何, 怎么样

```s
京都はどうでしたか
とても　綺麗でした
```

## 副助词は & 格助词が

### 副助词は
它作为助词的时候发音会发生变化 要读 wa

```
A　は　B　です
```

<br>

### 作用1:
提示助词, **提示A部分是全文的主题**

主题的概念是比主语要大的 主语的概念是包含在主题里面的

<br>

### 中日表达上

**中文: 主谓宾**
```s
主语   谓语  宾语
 ↓     ↓     ↓ 
小李   是   中国人
```

<br>

**日文: 主宾谓**
```s
 主语    宾语   谓语
  ↓       ↓     ↓ 
李さんは　中国人　です
```

<br>

### 理解:
は可是理解成 **呢**, 比如
```s
李さん　は　中国人　です

# 小李是中国人
# 小李呢, 是中国人
```

<br>

### は: 强调的是后面
は是提出一个主题, 然后后续的内容都是围绕着主题进行的

除了上面接在ではあります, 强调否定之外 它可以表示一句话的主题

强调的是 は后面的部分 **は(强调部分)**

<br>

### が: 强调的是前面
强调的是 が前面的部分 **(强调部分)が**

```js
// 提问的是谓语的部分
    ↓
你是谁？

// 在谓语这里是要使用 は 来进行提示, は后面的学生是我们要了解的新信息
私は学生だ



// 提供的部分在前面 也就是提问的部分在谓语的外面
↓
谁是学生？

// 所以要使用 が, 强调的是 が 前面的 私是我们了解的新信息
私が学生だ
```

<br>

同样是 我是学生 这句话 强调的部分不同 我们会分别使用 副助词は 和 格助词 が

- は强调的是 は后面的部分
- が强调的是 が前面的部分

<br>

### 作用2: 表示对比
```s
小野さんは　今日は　休みます

# に・て・から・まで・と　＋　は
私の部屋には　電話がありせん
```

<br>

### 主题 和 主语

**主题:**  
将一个话题 提出来 后面的内容是围绕话题所展开叙述的

<br>

**主语:**  
有的时候主语也可以叫做主题, 但是主题并不总是主语
```s
# 李さん既是主语也是整个句子的主题
李さんは　北京ダックを食べました
```

<br>

### は和が一起使用时
1. 整体 和 部分
```s
# 饺子是整体 它的局部有皮, 有馅, 这里将皮单独的拿出来说
て作りの餃子は　皮が　美味しいですよね
```

<br>

### 副助词も
它跟は的词性一样 在判断的基础上添加了 也 的意思
```js
// 使用的是は 所以强调的是学生
私は学生です。彼も学生です
```

## は　が 对比 
可以听下 16 19 课

之前我们说过 大助用 は, 小助用 が, 也说过疑问词做主语用 が

|は|が|
|:--|:--|
|主题|主语|
|对比, 强调|对象|
|所在|存在|
|已知|未知|
|判断|现象|

### 已知 & 未知
```s
A は　B　です

A: 是 旧信息
B: 是 新信息
```

疑问词在后我们用 は
```s
山田先生は　どの方ですか

    山田先生は　あのかたです
    あのかたが　山田先生です
```

## N名詞 + を + V动词
表示动作的对象或者内容 (宾语)

- 中文 宾语在后, 动词在前, 比如 听音乐, 吃苹果
- 日文 宾语在前 + 助词を + 动词

### 什么是宾语:
“宾语”是指在一个句子或短语中 **接受动作** 或 **作用** 的事物或人。

宾语通常是及物动词的对象, **表示动作的承受者或受益者**。在一个句子中, 动作的主体是主语, 而**动作的对象则是宾语**

<br><br>

## ます形:
ます型主要用来表示　肯定的 一般动作状态, 现在的动作状态 或 将来的动作状态 (叙述恒定真理, 现在的习惯性动作, 状态以及未来的动作, 状态)

1. 现在的习惯性的动作
2. 现在的动作和状态
3. 将来的动作和状态, 将来要发生的事情 行动, 行きます **翻译成我打算(计划)去**

也就是说一般现在时 和 将来时 都是使用 ます形 来表示的, 一般在表示将来时的时候, 一般带有要进行那个动作的意志

```s
太陽は毎日　東からのぼります (一般, 恒定)

毎朝、６時におきます (一般, 习惯)

明日、６時におきます (将来)
```

<br>

### ません:
现在 或 将来 的否定形式, 一般在表示将来时的时候, 一般带有要进行那个动作的意志, 否定形态的话 翻译成 **不打算**

<br>

### ました:
过去时, 过去时态一般表达如下的3种情况

1. 动作完了, 动作结束了

2. 过去时态, 昨天做了什么事情

3. 回忆和确认
```s
# 回忆 和 确认
今日泊まるホテルは天安飯店でしたね
```

<br>

### ませんでした:
过去没做某件事

<br>

```s
# 将 北京烤鸭 作为主题, 相当于我们这么说 北京烤鸭 你吃了么
```

### 注意:
我们只在具体的时间后面加上 に, 如 (包含数字的时间, 也就是可根据钟表, 日历上确定的)

如下的这些时间名词的后面一般不加 に, 下面的这些时间名词表达的含义是随着当前的时间变化的(也就是根据现在时间确定的时间), 如 今天到了明天就变成昨天了
- 今
- 毎日
- 明日
- 今日
- 先週
- 昨日

### 疑问词: いつ

它的后面不加 に, 当我们要询问具体的时间的时候 要使用 **何時、何曜日、何日**

<br>

**いつ 要点:**  
いつ后面接的动词是需要持续一段时间的, 比如 **展覧会は　いつから始まりますか** 开始之后这个展览会会持续一段时间, 如果我们这么说 就会很奇怪, **展覧会は　いつまで終わりますか**　我们会翻译为展览会结束到什么时候 有点奇怪 因为结束这个动作是一个瞬间性的动作, 结束了就是结束了 不会持续一段时间的

我们可以改成: **展覧会は　いつまでですか**

## あげる　もらう　くれる　等相关用法
用在物品互换 互赠的时候使用

<br>

### 授受动词
授: 表示给予, 受: 表示接受

- あげます  给
- もらいます  得到
- くれます  给我 (给我这一方的人)

上面三个动词是表示 授 与 受 之间的动词

<br>

### あげる: 给 / 给出
我 为对方做了某件事情, 翻译并不一定要用给, 只是方便记忆

あげる, 主要表示我给对方提供了一些事物 或者 信息等 类似这样有方向性的或者说供给性质的动词还有
1. あげる 给
2. 貸す 借给
3. 教える 教谁

<br>

**要点:**  
1. は的前面是主语, 我是提供者
2. に的前面是目标, 表示给谁 也就是接收者

**从给予者的角度表示 物/所有权 的转移 (都是从主语部分考虑)**  
![あげる方向.png](./imgs/あげる方向.png)

<br>

**使用场景:**  
表示物品的所有权由内向外的移动

- 我 -> 第二人称 或 第三人称
- 第二人称 -> 第三人称
- 第三人称 -> 第三人称

<br>

**注意:**  
1. 别人给我们则不能使用 あげる
2. N2(人)位置 **不能放** 我 或 我方人员 (亲属都相当于 我方人员)
3. 当给予的对方 比 自己的地位 **高** 的时候使用 ``差し上げます``
4. 当给予的对方 与 自己的地位 **平等** 的时候使用 ``あげます`` 
5. 当对方比自己的地位 **低** 的时候使用 ``やります`` 特别是给动物喂食 或 给植物浇水等

<br>

**敬语:**  
- 上: 差し上げる
- 下: やる 

<br>

**语法: N1(人)は N2(人)に N3(物)を あげる**   
```s
私は友達にプレゼントをあげた。

小野さんは友達に本を貸した。

私は子どもに英語を教えています

私は課長にプレゼントを差し上げます
```

<br>

### もらう: 收到 / 得到
我 得到了 某些事物 或者是 信息, 翻译并不一定要用收到, 只是方便记忆

类似这样有方向性的或者说供给性质的动词还有
1. もらう 收到
2. 借りる 借进
3. 習う　 学习

<br>

**要点:**  
1. 主语是我, 我是接收者
2. 人物に, 这里的人物指的是 **给予者**
```s
山田さんにはなをもらいました # 收到了 山田 的花

香里奈さんにCDを借りました # 从 他那里 借来了 CD

わんさんに中国語を習いました # 我 跟 王 学习中文
```

3. 当提供者不是个人而是机构的时候 我们要使用 から
```s
銀行からお金を借りました # 我跟银行贷了款
```

**从接收者的角度 表示 物/所有权 的转移 (都是从主语部分考虑)**  

<br>

**使用场景:**  
- 我 <- 第二人称 或 第三人称, 我从别人那里得到
- 第二人称 <- 第三人称
- 第三人称 <- 第三人称

<br>

**注意:**  
1. 当第三人称的其中之一是说话人的亲戚的时候, 按说话人的立场处理
```s
母は　小野さんに　ハンカチを　もらいました
```

2. 与 あげる 的句型不同, 在此句型中, 对给予者(B) 没有严格的身份限制(不能放我方人员的限制), 但一般不说 ``〜は　私に　〜をもらう``

3. 从比自己地位 **高** 的人那里得到的时候, 要用 ``いただきます``
4. 与自己平级 或 比自己地位 **低** 的人时, 要用 ``もらいます``

<br>

**敬语:**  
- 上: いただく
- 下: もらう 

<br>

**语法: N1(人)は N2(人)に　N3(物)を もらう**   
A从B处得到~

<br>

**应用:**   
``もらう`` 还可以表示接到了某人的电话, もらう前面是通信手段
```s
# 接到了长岛的电话
長島さんに電話をもらいました

# 接到了谁的电话
〜に・から　電話・ファックス・メール・手紙を　もらいます

# 给谁打电话
〜に 電話を します・かけます
```

<br>

### くれる: 别人给我
上面的 あげる 不能用于, 别人给我 或者 别人给己方家属等情况, 当我们想表达别人给己方的时候 我们就要使用 くれる
```s
佐藤さんは 私に はなをくれました
```

因为该句型一定是别人给我, 所以很多场景中, 把 ``私に`` 省略了
```s
ばさんは　私に　地図をくれました

ばさんは　地図をくれました
```

**要点:**  
1. に的前面 表示对象, 这个句型中 对象一定是自己 

<br>

**使用场景:**  
- 第三人称 -> 我, 别人给我
- 第二人称 -> 我, 别人给我

```s
# 对象语是妹妹, 因为妹妹是我这一方的人
昨日、馬さんが　妹に　旅行のお土産をくれました
```

<br>

**注意:**  
1. 对方比自己年长 或 地位高 的时候, 使用 ``くださいます``
2. 对方与自己平辈 或 平级的时候, 使用 ``くれます``
3. 日语中一般采用以说话人做为主语进行叙述的说法, 因此 与 ``馬さんは　私に　葉書をくれました`` 相比, 日语常说 ``私は　馬さんから　葉書をもらいました``

<br>

**敬语:**  
- 上: くださる
- 下: くれる 

<br>

### 总结:
上面的3个句子是用在物品互换 互赠的时候使用, 而下面加て形的三种用法, 首先是表达动作方向, 同时表达受惠或使对方受惠

比如下面的句子, **本质上就是 本を貸す, 翻译是也正常翻译** 但是加上了 てあげしまた 就表示 受惠或使对方受惠 的情况
```s
私は木村さんに本を貸してあげしまた
```

## て形 あげる・もらう・くれる
这里表示 谁给谁干什么事情 或 谁给谁做什么事情

上面我们表示的授受关系是物, 这里我们的授受关系体现在动作, 这里的动作都是主体的动作 (该组句型有恩惠的意思)

<br>

### て形 あげる: 帮谁做什么 / 给谁什么 / 什么动作 给谁
**强调说话人为 他人提供某种帮助或好处 或 说话人为 别人 做某事**, 什么样的好处是根据动词来的

当出现**动词てあげる**时, 表达了说话人做某事为别人带来利益或好处的意愿或行动。这强调的是说话人主动为他人提供帮助或好处。

动词てあげる　**这个动作是自己做**

```s
手紙を書いてあげる。（我给他写信。）

# てあげる: 帮着拿, 森帮年长的人拿行李了
森さんは　お年寄りの荷物を　持ってあげました
```

- 教给谁: 教えてあげる
- 读给谁: 読んであげる
- 介绍给谁: 紹介してあげる

```s
私は　王さんに　日本語を教えてあげる

中村さんは　桂子さんに　仕事を紹介してあげる

私は　ねこに　餌を飼ってやります
```

<br>

### て形 もらう: 主语得到 谁 帮助 / 主语请 谁 帮助 / 得到别人做的某事
もらう本身就是 得到 的意思, 有点被动的感觉, 有种**恩惠**的语感

**强调说话人从他人那里接受帮助或好处。**, 什么样的好处是根据动词来的

当出现**动词てもらう**时, 这表达了说话人从别人那里接受帮助或好处。这强调的是说话人作为接受方, 从他人那获得了帮助或好处。

动词てもらう　**这个动作是对方做**, 不是自己做哦

```s
友達に宿題を手伝ってもらった。（朋友帮助我做了作业。）

# 森请小李 案内 北京
森さんは　李さんに　北京を案内してもらいました
```

<br>

### て形 くれる: 给我 / 帮我 做了什么 / 别人为我做了某事
**也是强调从他人那里接受帮助或好处**, 但更强调施惠者的好意和行为, 常用于描述亲近人之间的行为。有点主动的感觉, 有种**感恩**的感觉

当出现**动词てくれる**时, 这也是表达说话人从别人那里接受帮助或好处, 但与もらう相比, くれる更强调施惠者的行为和好意, 通常用于亲近的人之间。

动词てくれる　**这个动作是对方做**, 不是自己做哦
```s
友達が　私に　本を貸してくれた。（朋友借书给我了。）

# 一个女人帮我捡起了钱包
女の人が　私の財布を　拾ってくれました
```

这个句型中 主语的部分一般都会使用 が

<br>

**扩展: て形 くれませんか？这个句子作为疑问形式的时候, 用于委托关系亲密的人为自己做某事**  
```s
# 委托别人帮自己做某事的时候, 使用否定形式　てくれませんか 比使用 てくれますか 要更加的客气
森さん、この手紙をコピーしてくれませんか
```

## 形容词:

### 形容词的作用:
用于描述 某人 或 事物 **的性质**  

形容词既可以做谓语 来描述主题的性质, 也可以直接修饰名词作定语

```s
东京很繁华 (形容词谓语)

```

- 2类形容词: 以 な 结尾的词

<br>

### い 形容词谓语句
形容词作谓语的时候, 时态的体现不是靠 です 而是靠形容词本身的时态

**N + は + い形 + です**  
```s
四川料理は　辛いです
```

<br>

### い 形容词 注意点:
1. 多い　と　少ない、它们**一般**不会修饰名词 多直接用做谓语 
```s
多い本　×
たくさんの本　○

週末の公演は　人が多いです　○
```

<br>

### な 形容词谓语句
```s
京都の紅葉は　有名です
```

<br>

### な 形容词 做定语
```s
ならは　静かなまちです

# 专有名词解释: 词干 静か
```

## N(场所/时间 范围) + 中

### 场所 + 中 (じゅう)
表示其场所范围内的全部

<br>

### 时间/期间 + 中 (じゅう)
表示在某个期间一直

## 形容词 + の
也叫做 准体助词, 用来代替前面出现过的名字, 整个被当做名词部分来处理

- い形容词 + の
- な形容词 + な + の


### 语法1: N1は N2が　い形容词 / な形容词 です
某人 对 某物 具有 什么样的 情感 (一般用于表达自己的内心感情)

- N1: 具有情感的主体, 用 は 来表示
- N2: 情感的对象, 用 が 来表示, 比如喜欢的对象, 讨厌什么
- 谓语部分: 表示什么情感的形容词

```s
# おのさん 主体
# 歌 情感的对象
# 好き 情感的形容词
おのさんは　歌が　好きです
```

<br>

**注意:**   
我们使用这个句型的时候, 一般表达的是自己的感受, 所以

**N1**通常是第一人称 我, 而一般不会是第三人称 おのさん, 疑问句形式用于第二人称 (あなた) 

因为在日语中一班不会直接说 **对方** 或 **第三者** 的情感 和 内心的感受, **日语中一般直接说第三方的情感是失利的**

上面不是语法的问题, 而是文化方面的问题

<br>

### 语法2: N1は N2が　能力V词 / い形容词 / な形容词 です
表示能力的句型

谓语部分是表示 能力 的词, 表示能力的对象怎么样
- わかります
- できます
- 上手です (水平高)
- 下手です
- 苦手です (不善于)

<br>

- N1是 能力的主体
- N2是 能力的对象

<br>

**提示:**  
该句型的主体部分可以是第三人称
```s
Erinは日本語が上手です
```

<br>

### 技巧: 苦手 经常表达不喜欢什么东西
在日语中我们表达不喜欢什么的时候, 一般不会直接说 きらい, 而是会用 **にがて** 它在日常生活中表达不太喜欢的意思  

```s
我不太喜欢吃鱼 (私は魚が苦手です、李さんが苦手です)
```

## 〜から、小句
から 接在表示理由, 原因的小句的句尾, 翻译: 因为...

通常是先陈述理由, 然后陈述结论, 但有的时候也会相反
```s
# 一个句子
寒いですから、窓を閉めます

# 两个句子
まどを閉めます。寒いですから
```

### だから・ですから
**独立**使用在表示结果, 结论的句子的**前面**, 翻译: 所以...
```s
# 两个句子, 前面的句子很完成使用句号进行了结句, だから后面又是一个新的句子
きむさんは野菜が嫌いです。だから・ですから、あまり　野菜をたべません
```

## 比较句

### N1**は** N2**より** い形容词 / な形容词です
N1 比 N2 更 ...

**より:**  
~より, 比~

它是比较的基准

```s
李さんは　森さんより　若いです

```

<br>

**注意:**  
该句型必须以 **肯定形式** 结句
```s
# 错误演示
李さんは　森さんより　若くないです
李さんは　森さんより　若くありません
```

<br>

### N1**より** N2**のほうが** い形容词 / な形容词です
跟N1比较起来(比起日本来), N2更加的 ...

**より:**  
~より, 比~

它是比较的基准

```s
大阪より　東京のほうが　にぎやかですか

紅茶より　コーヒーのほうが　好きです
```

<br>

**惯用句:**  
表达我喜欢什么 / 我想要什么, 我觉得什么比较好
```s
〜のほうが　いいです

# 想要钱
おかねがいいです
```

<br>

### N1**は** N2**ほど** い形容词・くない / な形容词・ではありません
N1 不如 N2 那么 ...

**注意:**  
句尾必须使用否定形式

**ほど:**  
~ほど, 表示程度  

```s
# 神戸 没有 大阪 那么繁华
神戸は　大阪ほど　にぎやかではありません

東京の冬は　北京の冬ほど　寒くないです
```

<br>

**没有比 N1 更加 怎样么的了:**  
```s
富士山ほど、綺麗な山がないです
```  

<br>

### N1と N2と　どちらが い形容词 / な形容词ですか
N1 和 N2 哪个更 ...

**回答:**  
1. 〜の　ほうが　い形容词 / な形容词です
2. どちらも　い形容词 / な形容词です

<br>

### N1の中で N2が 一番 い形容词 / な形容词ですか

**で:**

**注意:**  
当N1表示 场所 或 时间, 则后面不加 **の中**, 而是直接使用 **N1(場所・時間)で**, 因为表示场所 和 时间的词本身就有范围的含义了 所以直接加で

## て形 相关句型
て形的基本用户, 主要用来连接前后 两个动词 或 两个句节

用 て形 连接的两个动词 一般前后两个动作 或 事情之间存在
1. 相继发生
2. 并列
3. 手段
4. 弱因果的关系 (因为所以)
5. 前者为后者的伴随状况

书上在初级部分 就使用前后发生的用法, 也就是在时间上按照前后顺序发生
```s
昨日　デパートへ　行って、買い物しました
```

<br>

本来动词て形在句型中是可以重复出现的 但是有的时候连接多了, 会使人觉得难以理解 而是会觉得花很啰嗦, 所以一般只会有て形连接两到三项 如果需要连接更多的动作 则可以把它们算开构成两个句子中间使用 それから 进行连接

<br>

### 动词て形 动词て形 动词
**1. 列举动作或状态**  
当你想要列举两个或更多的动作或状态时, 可以使用动词的て形。这类似于英语中的"and", 用于连接句子或短语

```s
図書館で本を読んで、勉強して、友達に会いました。(我在图书馆读书、学习、并见了朋友。)
```

<br>

**2. 动作的顺序(相继发生)**  
表达一系列动作按照一定顺序发生时, 可以使用动词的て形来连接这些动作, 表明它们是依次发生的

```s
朝起きて、顔を洗って、朝ご飯を食べました。(早上起来, 洗脸, 然后吃早饭)

昨日　デパートへ　行って、買い物しました。
```

<br>

**3. 并列关系**
```s
太郎は　留学して、次郎は　東大に入りました
```

<br>

**4. 手段**  
```s
新聞紙を使って、紙飛行機を　作りました

今日は歩いて　帰ります
```

<br>

**4. 弱因果: 小句1で形, 小句2**  
它的因果关系不像から那么强 但是前后之间还是有那么点因果关系的
```s
あのビルは　エレベーターが　あって、便利です
```

其中小句1表示原因, 但是不强调韵苑, 一般 ``'因为'`` 两个字不翻译出来, **它主要强调的是前面的原因, 自然地导致了后面的结果**

```s
# 我迟到了, 真抱歉
遅くなって、すみません

# 开始的时候语言不通, 非常困难
最初は言葉が通じなくて、とても困りました

# 解释的非常好, 我完全清楚了
説明が上手で、よく分かりました
```

补充:
1. から 偏主观
2. ので 客观
3. て形, 

<br>

**5. 伴随状况**  
前项是后项的伴随状态
```s
# 穿着新衣服 参见晚会
新しい洋服を着て　パーティーに行きました

暑かったから、窓を　開けて　寝ました
```

<br>

**6. 处于...的状态, 做的某事:**  
1. 小句1 て、小句2
1. 小句1 ないで (ずに, ないで可以替换成ずに)、小句2

```s
# 带着眼镜看书
メガネをかけて、本を読みます

# 举着手过马路
てをあげて、横断歩道を渡ります

# 今天没有带伞就出门了
今日はかさを持たずに、出かけました
今日はかさを持たないで、出かけました

# しないで -> せずに
勉強せずに、テストに参加します
```

注意: 该语法的前后主语必须一致

<br>

### 动词てから、动词
翻译为 先怎么样, 然后怎么样, 前面的行为为前提或者是准备动作, 或者 **...之后...**

表示前后两个动作的相继发生, 这里是不是跟上面说的 て形 的相继发生有共同之处, 确实这个句型的语义和上面 て形 的相继发生 用法相同, **但是 动词てから 不能在一个句子中反复使用两次以上**

最大的不同之处在于, **Aてから, B**, 强调 **A和B先后顺序不可逆**, 而我们单纯的使用 て形 连接的两个动作就未必是先发生前项再发生后项

```s
りさんは　毎晩　ラジオを聴いてから、寝ます

# 先吃饭 后吃药
医者：この薬はご飯を食べてから、飲んでください。
患者：あっ、今まで、飲んでから、食べていました。
```

<br>

翻译为: 请怎么样 (请求对方做某事)

<br>

### 动词て形 います
1. **动作现在进行时** 这时的动词通常是带有持续性的动作或变化
```s
小野さんは　今　新聞を読んでいます
```

2. **表示状态**, 当 てます 前面的动词接的是瞬间性动作或变化动词的时候, 这时它表示动作或变化结束后 留下的结果状态
  - **瞬间性动词(知ります 死にます):** 这里就是瞬间性动词结束后留下的结果状态
  - **变化的动词:** 太ります就是变化的词 当它表示状态的时候 就用 太ってます 体现变胖后的状态
```s
私は結婚してます　# 我结婚了

私は山田さんを知ってます　# 我认识山田

私はカメラを持ってます　# 我有相机, 这里持ってます有两种意思 1. 拿在手里, 2. 拥有 的意思
```

3. **习惯性的行为**, 职业等, 注意: 一些风俗习惯不含在内
```s
スーパーでナンプラーを売ってます

# 错误的演示 (一些风俗习惯不含在内, 印度用手, 美国用刀叉)
日本人はお箸でご飯を食べてます
```

4. **表示反复**
```s
# 每天都是这样
李さんは毎日学校に通いながら、アルバイトしてます
```

<br>

## 他动词てあります
表示为了某种目的, 进行的某一种动作, 其结果或状态仍然存在

用来表示某个动作已经被完成, 并且结果持续到现在, 说明某个状态是由过去的某个行为造成的, 并且这个状态或结果至今仍然有效或存在。  

接他动词之后, 表示某人行为动作的结果留下的某种状态。根据上下文的关系, 有时也可带有"为将来做好某事"的意思。

**有点事先预先的意思, 我做这件事情的时候就是为了某件事准备的**

<br>

**使用场景:**  
1. 描述物品的制作或准备状态
2. 表明某个安排或决定已经被做出

<br>

**区别:**  
这个结构强调的是动作的完成和其结果的持续性。它与"ています"不同
- "ています"更多用于表示动作的进行状态或习惯性动作。
- "てあります"主要用于他动词, 强调某个动作被某人完成, 而这个动作的结果持续到现在。

## 欲望 和 愿望 的表达方式: ほしい　と　ます形＋たい

### N1(人) は N2(物) がほしい
翻译为 **想要**, 表示想得到某物的愿望

ほしい是1类形容词

句型中
- N1 为 愿望的主体 (主语是说话人 或者是 听话人)
- N2 为 愿望的对象

```s
私が お金がほしいです
宮田さんは 本がほしいですか
```

<br>

**注意:**  
1. ほしい 是表达感情的形容词, 感情所指向的对象需要使用 助词 が

2. 当为 陈述句时 只能用于说话人描述自己的愿望 (不能用于表达你想要什么 或者 他想要什么)
```s
あなたは　新しい洋服がほしいです　❎
```

3. 当为 疑问句时 可以用于第二人称 你想要什么?
```s
宮田さんは 何がほしいですか
```

4. 第三人称不能用于此句型

<br>

### V-ます形＋たい
翻译为 **想干什么**, 用于表示第一人称的愿望, 通常用于表达说话者自己的愿望或欲望 我想干什么, 比如

1. 行きたい 想去
2. やりたい 想做

```s
私は日本に行きたいです。 (我想去日本。)

何を食べたいですか。 (您想吃什么？)
```

<br>

**注意点:**  
1. 当我们将动词变为たい形式后,原句型中的 を 可以换成 が, 两个助词都可以, 但要注意
   ```s
   何　を・が　食べたいですか。 (您想吃什么？)
   ```
  1. 如果语句中有其它助词, 则不要换成が, 例如に・と・へ这些都不能换 只有を可以
  2. 当语句中的助词 を 表示的是离开或经过的场所时, 则不能换成 が, 只有表示宾语动作的对象的を可以换成 が
  ```s
  いえを出たいです　✅
  いえが出たいです　❎
  ```

## 变化的表达
在日文中只要是状态发生变化, 就可以翻译成 变

<br>

### 场景:
1. 电视声音小, 我们扭动旋钮, 将声音调大
2. 下雨了, 越下越大

上面的两个场景的区别是什么? 一个是雨变大了, 一个是声音变大了, 在中文中两个场景都是变

但是上面两个场景的聚焦点是不一样的

场景1中的聚焦点是人为动作加入, 使一些事情发生了变化, 我们还可以将聚焦点放在, 人扭动旋钮之后, 声音变大了, 聚焦点为声音变大后的结果和状态

场景2中的雨变大了, 强调的是雨自身的行为 雨变大了

<br>

### なる　と　する　总结:
也就是我们的聚焦点不同, 我们选取的动词也是不同的

- 如果我们强调的是 **自然而然的变化(自身的变化)** / **变化的结果和状态** -> 选择 なる (自动词)

- 如果我们聚焦点是 **人为使什么发生了变化** -> 选择 する (他动词)

<br>

- い -> く + V
- な -> に + V
- N -> に + V

## よね

### よ
よ是一种偏单向的

1. 断定, 主张, 叮嘱, 提醒 对方注意, 一般翻译为: 哦
```s
あなたが行かなくても、私も行くよ
```
2. 强调语气
```s
一緒に、勉強しましょうよ
```

<br>

### ね
ね是一种双向的感觉, 当我们在进行感慨的时候, 比如今天天儿真好哈

1. 感叹 (降调)
```s
ずいぶん、綺麗な部屋ですね
```

2. 征询: 希望对方同意自己 (升调)
```s
このケーキが美味しいですね　(升调)
```

3. 同意: 表明自己同意对方的观点
```s
そうですね
```

4. 轻微的主张, 比如 我认为我是对的呢(ね)

<br>

### よね
它是终助词的叠加, 意思为其叠加之后的含义, 判断句子语气, 以最后一个终助词为标准

它既有よ的意思, 也有ね的意思

一般用在 **对自己的意见没有足够的把握的时候使用**, 因为不确定对方是否 100% 同意自己的观点
- 如果我确信 你会同意我的观点时 就使用 ね 了
- 如果不确信 就使用 よね, 我先说自己的观点, 然后征求同意的感觉

```s
このケーキが美味しいよね
# よ: 表示自己的主张
# ね: 征求对方的同意

# 断句: このケーキが美味しいよ, ね, 表示我先提出自己的观点 然后去征求你的意见

# 我的记忆出现差错 不知道是明天还是后天了 我跟对方确认下
李さんの送別会は　明日ですよね
```

<br>

### か
1. か放在句末时有如下的意思: 疑问, 反问, 确认, 理解 (降调, なるほど、宮田さんですか)

2. か放在句中: 表示 不确定 (当か放在句中时后续有 はがを 的时候 它们可以省略)
  1. 疑问词 + か: いつか、誰か 
  2. 简体句 + か: 表示 不确定

<br>

**情况1: か前面的小句(简体句)中 有 疑问词的情况**   
该情况为 一个长句中的一部分中包含疑问词的情况

比如 我不知道什么时候去

当小句中有疑问词的时候, 我们需要将这个小句变成简体的疑问句 再加　か

```s
# 钥匙在哪里? + 教えてください。
鍵がどこにあるか、教えてください。

# 什么时候去日本? + まだ決めていません。
いつ日本へ行くか、まだ決めていません。

いつ結婚するか、教えてください。

誰が来たか、知っていますか
```

<br>

**情况2: か前面的小句(简体句)中 没有 疑问词的情况**  
比如 我不知道去还是不去

句型: 这里表现的是 动作的不确定性, 去? 还是 不去?, 有选择的意思
```s
〜か、〜ないか

# 不确定 去 还是 不去
行くか、行かないか

# 不确定 高 还是 不高
高いか、高くないか

# N词虽然是简体 但是去掉 だ
学生か、学生ではないか
```

注意1: 我们可以将 ``〜ないか`` 的部分 替换成 ``どうか``, 整体句型就变成了 **简体句 + かどうか**

注意2: 名词 和 形容动词 的简体句 直接加　か・かどうか 后者理解为 **去掉だ**加　か・かどうか
```s
行くかどうか

私は　ことしの夏に北京へ行くか、行かないか わからないです
私は　ことしの夏に北京へ行くかどうか、わからないです


# 私は 去年森去没去过日本 わかりません
私は　森さんが去年日本へ行ったか、行かなかったか わかりません

# 1. 从句中的主语使用 が
# 2. 从句中的时态是过去, 但整体的时态语境是现在说的话
私は　森さんが去年日本へ行ったかどうか わかりません


# 2类形容词 和 N词 接かどうか的时候 中间没有任何东西哦
週末が暇かどうか わかりません
週末が暇か、暇ではないか わかりません

彼が有名かどうか聞いたことがありませんか

森さんが学生かどうか知っていますか
```

**翻译: 是否**

## 过去形 (た形)的 基本应用

### 变化规律:
等于 て形

<br>

### 概念:
动词的た形 也叫做 完了 助动词, 它表示如下的概念

1. 表示动作的 **完了 和 结束**
2. 搭配时间, 表示 **过去的动作 或 状态**
```s
先月　日本へ　いった
```

<br>

### 应用1: た形 + ことがある
表示 曾经有过某经历 或 经验, 一般 **不用于近期** **普遍** 会发生的事情

这个时间一般来说只要你觉得是曾经就可以, 但是上周的话就不会用这个句型

翻译: 我之前(曾经)做过什么

**错误表达:**  
```s
# 错误表达: 不用于近期 - 昨天曾经去了日本
昨日、日本へいったことがある


# 错误表达: 普遍 - 我曾经感冒过
風邪を引いたことがある
```

<br>

### 应用2: た形 + ことがない
表示之前(曾经)没有做过什么, 没有上面的特殊要求, 属于上面句型的否定式

<br>

### 应用3: Nの・た形 + あとで、〜
表示在做完前项的动作后, 做后项的动作

翻译: 在...之后

下面说这句话的时候 还没有下课, 属于对将来的某件事情的建议, 所以**这里面的 た形 体现的不是过去时态 而是动作的完了**
```s
授業が終わったあとで、カラオケに行きませんか
``` 

<br>

**前に・後で: 时间状语从句:**  

1. 相对时态
2. 绝对时态: 句末的谓语动词决定整个句子的时态

我们时间状语从句中体现的是 相对时态, 也就是动作的先后关系

```s
# 1. 下面的句子表示的现在进行时, 当我们在进行洗手这个动作的时候, 吃饭这件事情还没有发生
# 2. A 前に B, B先发生A是将来要发生的动作, A是未发生的动作
# 这里的 食事をする 动词原形表示将来, 这里就是相对时态, 相对于洗手 它是将来发生的事儿
食事をする前に
てを洗ってください

现在    洗手  吃饭
↓       ↓     ↓
|_______|_____|



# A 後で B, A动作发生后, 再做B
# 这里的 下课 和 唱歌 就是相对时态, 相对于主句唱歌这个动作, 下课的动作已经发生完了
授業が終わった後で、
カラオケに行きませんか

现在    下课  唱歌
↓       ↓     ↓
|_______|_____|
```

<br>

**後で・てから 的区别:**  
它们都是之后, 区别如下

1. 前后两项动作是**同一个主体**发出的话, 且是**有意安排时间**的行为, 後で・てから都可以
  - てから: 动作时间 间隔 较短
  - 後で: 动作时间 间隔 较长
  ```s
  # 都可以
  ご飯を食べてから、散歩に行く
  ご飯を食べた後で、散歩に行く
  ```

2. 前后两项动作 主体不同时, 
  - 若两个动作为 有意安排 多用 てから
  - 若后项为 偶然发生 的动作时 多用 後で
  ```s
  # てから
  あめがやんでから、散歩に行く

  # 後で
  ご飯を食べた後で、友達が来た
  ```

3. てから 可以表示在一段时间内 一直在做, 或 处于 某种状态, 後で 则表示该动作完全结束后
```s
# 一段时间内 处于 某种状态: 上车之后开始恶心
車に乗ってから、気持ちが悪くなった

# 表示该动作完全结束: 上车然后下车后 才恶心, 乗った後で 表示上完车了 它表示动作完全结束, 等我们从车上下来了后 变的不舒服了
車に乗った後で、気持ちが悪くなった
```

4. てから 可以表示 时间的流逝 和 经过
```s
# 这里就不能使用 後で
日本に来てから、三ヶ月になります
```

<br>

### 应用4: 体言・基本形・ない形・Vた形  ほうがいいです
翻译: 
- ない形ほうがいい: 还是不要做某件事情比较好
- 基本形ほうがいい: **客观的 普遍道理**, 不掺杂主观情绪, 翻译: 还是做某件事情比较好
- Vた形ほうがいい: **主观的 建议**, 建议感比较强, 翻译: 还是做某件事情比较好

用于向听话人提出建议和忠告的时候使用, 包含将两者加以比较, 选择后者比较好的意思

```s

よく日本語を練習した方がいいですよ

野菜や果物を食べる方がいいですよ
```

<br>

**扩展:**  
如果是单纯的建议可以使用 **〜たらいいです**  
```s
お寺を見たいんですが

京都へ行ったらいいですよ
```

## 应用6: 动词た形 + ばかりです
表示某动作或某事情 发生之后还没有过多久 这种心情, 不管实际上经过的时间长短, 主要强调是说话人觉得时间短就可以

### 应用7: 谓语的举例: 〜たり、〜たり、〜
比如有人问我们 周末 都干什么, 我们会回答 看书 写字 打游戏等

这里我们说的例子都是动词也就是谓语部分, 我们N词, 形容词, 动词都可以做谓语

<br>

**接续:**  
1. 动词: Vたり、Vたり、する (前面是动词たり时, 我们使用 **する** 结句)

2. 名词: だったり、だったり、だ (前面是名词だったり时, 我们使用 **だ** 结句)

3. い形容词: かったり、かったり、する/だ (前面是形容词〜たり时, 我们可以使用 **だ 或者 する** 结句)

4. な形容词: だったり、だったり、する/だ (前面是形容词〜たり时, 我们可以使用 **だ 或者 する** 结句)

<br>

**翻译:**  
1. 动词: 有时...有时..., 又...又..., 或者...或者...
2. 形容词: 时而...时而... (形容词通常为反义词)
3. 名词: 既有...也有...

这里注意我们列举的动作, 不是同时发生的, 是有时 或者 时而的感觉

<br>

```s
# 前面使用的是 毎日 表达式的反复的行为 我们使用的是 します
# 如果前面使用的是 昨天, 那么我们词尾需要使用 しました
毎日、散歩したり、買い物に行ったり、します

# を表示移动行为经过的场所, 表示在房间中走来走去
部屋の中を行ったり、来たり、する

# して表示句中的中顿, 或者是弱因果
毎日、学校に行ったり、図書館へ行ったりして、大変だ

昨日、映画を見たり、美味しいものを食べたりして、とても楽しかったです

# 形容词 结尾的时候 这里使用的是 だ: 这周边时而安静 时而热闹
この辺りは静かだったり、にぎやかだったり　です

日によって、忙しかったり、暇だったり　です

# 名词的时候 我们翻译为  既有...也有...
日本語の先生は中国人だったり、日本人だったりです
```

<br>

**注意:**  
1. 这个句型中仅仅是形式上借助了 た形, 含义上与其无关, 时态体现在句末谓语部分
2. 形容词 和 形容动词 句中的例子一般互为反义
3. たり可以只举一例, 暗指其他 (私は暇な時に散歩したりする)

## 後で、後に、後 的区别

### 後で
完成前项状态下, 进行后项

<br>

### 後に
强调时间点 (前后 间隔短)

在做完前项的时间点后, 我们做另外一件事情, 这个时间的间隔很短, 一件事做完了 过会 做另外一件事

<br>

### 後
后项多为 某种状态的持续, 比如
```s
両親が亡くなった後、彼がずっと独身だった
```

## と思う 系列

### と思う 系列 时态:

- と　思いました: 我过去想了什么

- と　思っています: 一直想

### と思う 系列使用人称的问题:
1. 第一人称都可以使用

2. 问的时候可以使用第二人称
```s
- と　思います ✅
- と　思いません ✅
- と　思いました ✅
- と　思いませんでした ✅

- と　思っています ✅
```

3. 第三人称
```s
# 我们没有办法知道第三人称现在的想法
- と　思います ❌
- と　思いません ❌

# 这里之所以能用 是因为过去时态, 这个想的内容过去发生过, 我们已经知道了 所以可以使用
- と　思いました ✅
- と　思いませんでした ✅
- と　思っています ✅
```

### と思う
翻译为 我想 我认为 我觉得, 我**现在**想的内容, 思考的内容, と前面是想的内容

```s
明日は　いい天気だ　と思います

日本語が簡単だ　と思います

# 我想小李马上就会来
李さんはもうすぐ来る　と思います
```

<br>

### と思った
表示过去的观点或想法
```s
# 我之前觉得这本书有意思
この本が面白い　と思いました
```

<br>

**示例:**  
1. 表达个人意见或感想
```s
明日は雨が降ると思います。(我认为明天会下雨。)
```

2. 表达决定或打算
```s
もう帰ると思います。(我想我要回去了。)
```

3. 推测或猜想
```s
彼はもう睡んでいると思う。(我猜他已经睡了。)

そのニュースは本当だと思いますか？(你认为那条新闻是真的吗？)
```

<br>

### と思っている

**回顾: っている的几种用法**  
1. 动作进行时
2. 结果 状态的持续
3. 反复习惯性动作

<br>

と思っている这里的用法就是上面的第3点, 反复习惯性动作, 翻译为: **一直想**

```s
# と思っています　一直在想
# と思っていました　之前一直在想
王さんは　ずっと前から　アメリカへ留学に行く　と思っていました
```

<br>

表示在说话之前就已经有这种想法了, 而且现在依然是这样认为的, 含有"早就想好, 计划好, 以前就有这种打算"的意思。

```s
李さんは日本語が難しいと思っています。 # 第三人称
```

```s
# 日本人习惯每天早上说 おはよう
```

- と　言いません: **不习惯**于说的内容

```s

```

<br>

### 接续:
说的话, 我们是可以 直接引用 或 间接引用 的
- 直接引用: B 将 A 说的事情完整的转述给 C
- 间接引用: B 将 A 说的事情 换成B自己的话转述给 C

```s

# 间接引用: 将 A 说的话 总结下 进行转述 使用 と
先生は　宿題が今週中に提出すると　言いました
```

## んだ・のだ　相关用法
接在连体形的后面, 来表示**加强语气(关心好奇/解释说明/个人主张/描述情况/请求提问的开场白等)**

说话人对见到和听到的事情 要求更为详细的说明 或者 要求就其理由进行说明时

<br>

### 接续
- 动词, 形容词 - 简体形
- 形容动词 词干 或 体言(N) + な, 如: 綺麗なんです

<br>

```s
# 客观描述 正在下雨
雨が降ってる

# 哎呀妈 下雨了
雨が降ってるんだ

# 医生病人之间的对话, 因为医生不需要有好奇等情绪
どうしましたか？  
  熱があった

# 朋友之间的对话 因为朋友会关心和好奇
どうしたんですか
  熱があったんです

# 这个包真贵啊
このカバンが高いのですね

# 好奇
可愛い　どこで買ったんですか

# 说话自己的情况
今朝五時に起きたんですが

# 加强语气 说明自己的想法 观点 自己的情况
新宿駅に行きたいんですが、どうやって行きますか
```

<br>

### 注意: 说话时不要用了
若过多的使用 ``んだ・のだ`` 来**提问**的话, 会有种 咄咄逼人 的感觉 (尽量不要对 领导 使用该句型), 具体使用与否要视情况而定

```s
# 你想要什么?
何がほしいですか？

# 你到底想要什么?
何がほしいのですか？
```

## 终助词 かな

1. 当有两个人, 一问一答的时候, 表达询问和不确定

```s

  ３時間かな
```

## 假定形
一般翻译: 如果...就... / 要想...就...

<br>

### 变形
1. 五段动词, 将 い段 -> え段 + ば
2. 一段动词 + れば
3. します -> すれば、きます -> くれば
4. 名词 和 な形容词 + なら
5. い形容词 去掉い + ければ: 辛い -> 辛ければ

<br>

### 句型: 假定形小句1、小句2
要想实现 小句2 的话, 必须让小句1先发生 (小句1是前提条件)

```s
ボタンを押せば、窓が開きます

この大会で優勝すれば、オリンピックに出場することができます
```

<br>

**注意:**  
原则上小句2 不能使用下列的表达方式

- 意志
- 希望
- 命令
- 请求

但是 **如果 小句1 的谓语为状态性谓语** 或者 **小句1 和 小句2 的主语不同的时候** 不受上面的限制

``状态性谓语:``   
指那些表示状态、性质、存在等的动词或形容词。

它们描述的是一种状态，而不是一个动作或变化。  
与动作性谓语（如动词的动作、变化等）不同，状态性谓语表示的是相对静态的、持续的状态。

常见的状态性谓语包括存在动词 **いる / ある**

表示性质或状态的形容词 **きれいだ / 高い**, 形容词多表示状态


```s
# 前后主语不一致, 所以小句2可以使用表示意志的动词
李さんはいけば、私も行く

# 形容词作为假定条件的话, 小句2可以使用意志形
私でよければ、やります

# あれば 状态性谓语, 小句2可以使用意志形
時間があれば、見に行きませんか
```

<br>

### 假定形 和 たら 的区别
当我们想表达 如果... 或者 要是... 的时候 多使用 たら 或者 ば

1. 〜たら 没有使用条件的限制
2. 〜ば 有使用条件的限制

**条件的限制1:**  
当小句1表示 **动作** 或 **变化** 的时候 (谓语部分为形容词或者ある・いる除外) 小句2不能是 たい 或者 てください 表示意志 请求等形式
```s
# たら的话 没有限制 ✅
雨が降ったら、窓を閉めてください

# 〜ば 则不可以 ❎
雨が降れば、窓を閉めてください
```

**条件的限制2:**  
〜たら后面的小句是过去时态, 表示发现了某种情况, 但是 〜ば 不行
```s
# 发现了某种情况 ✅
まどを開けたら、富士山が見えました

# 〜ば 则不可以 ❎
まどを開ければ、富士山が見えました
```

<br>

### 句型: 小句1(简体形)なら、小句2
如果...话, 就...

<br>

**接续:**  
1. 动词简体形 + なら
2. 形容词1 + なら: 忙しいなら
3. 形容词2 / 体言 + なら

<br>

**用法1:**  
观察对方的言谈举止, 推断出的假定 (来陈述自己的意见 或 想法, 以及向对方提出请求或忠告)

小句2多为 表示说话者主观立场的内容
- 判断
- 命令
- 提议

```s
刺身が嫌いなら、食べなくてもいいですよ
```

```s
天安門へ行くなら、地下鉄が便利ですよ
```

<br>

**用法2:**  
根据和对方的交谈, 提出某种结论 并给与对方意见
```s
今日は忙しいですが。。。
忙しいなら、行かなくてもいいですよ
```

<br>

**总结:**   
なら多用于以对方的某种意志或愿望为前提 而提出的某种建议, 这种用法 たら、ば、と都不具备

```s
新鮮な魚を買いたいですが。。。
魚なら、駅前のスーパがいいですよ
```

动词基本型＋「と」  相当于中文的"假如…""如果…"。

3. 然后再看子句的部分

# 定语从句
在一个长句子中找出定语成分

定语从句 我们要从下面的3个角度来考虑
1. 位置
2. 接续
3. 助词 

<br>

### 位置: 定语应该放在哪里?
日语中的语序特征为:
1. 谓语后置
2. 定语前置

定语是放在被修饰词之前的, 如 ``~样的 人``

<br>

### 接续: 定语接续: (〜)人
当作为名词的定语成分 接续方式如下
1. 体言 + の
2. い形容词本身
3. な形容词 + な (肯定情况下) / な形容词 简体形 (其他情况下使用简体形 如: 静かだったところ)
4. 动词简体形

<br>

### 助词:
主题 和 主语 最大的区别在于, ``主题 > 主语``

主语是: 主谓宾, 主语是谓语动词的执行者, 它是直接作用于谓语的, 从句中的主语叫做小主语(使用が)

主题是: 主语可以做主题, 任何动词都可以做主题, 比如 宾语, 正常是我吃饭, 我们还可以说饭我吃了, 我们想想这个饭是主语么? 不是主语吧 它在这句话中是 主谓宾, 我们在日语中将饭当成主题使用は来造句

<br>

**总结:**  
在定语从句中, 大主语该用谁就用谁, 小主语(定语从句中的主语)必须用　が　

<br>

```s
# 2. 看句末: 这是资料

あれは　倒産した　会社です

# 定语成分中出现了 小主语

```

## 形式体言:
形式体言也叫做形式名词, 是一种一般不具有实际意义的词, 其作用是能够使它前面的 **用言** 变成名词化, 从而使它们能在剧中做主语, 宾语等

<br>

### 形式体言一般有2个
- の: 代替 人 或 物
- こと: 代替 事情

一般情况下 の 和 こと 是可以互换的, 但是如果遇到以下的两点, 则它们之间不能互换

1. 当后续动词为 ``传达``, ``表达``, ``意志``, ``思考`` 以及 ``惯用句型时`` **只能用 こと**, 比如
  - 言う
  - 知らせる
  - 表す　
  - 示す
  - 望む
  - 決める
  - ことができる
  - ことがある

2. 当后续动词为表示 ``感觉`` ``知觉`` ``动作`` 的动词 **使用 の**　比如
  - 聞こえる
  - 見える
  - 待つ
  - 会う

总之
- 惯用表达方式除外, 在表示向别人传达的内容 或 决定的内容 以及 抽象的事情时 使用 **こと**
- 表示自己实际感受到什么 做什么等这些具体的行动 或 体验 时使用 **の**

**总结:**  
一般的动词我们可以使用 の, 但是我们要注意上面的必须使用 こと 的情况

## 〜時、〜: 时间状态从句
``時`` 是 体言, 翻译: 下面的句型都可以翻译为: ``...的时候, ...``, 但是在``時``前的接续上有时态的区别, 时态不一样, 表示的动作的先后关系也不一样

```s
A 時、B
```

这个句型分为两个部分, 前面的部分 和 后面的部分

<br>

### 动词基本型 + 時、〜
当 动词基本型 + 時, 表示**后项已经完成**, 但 **前项尚未开始**

表示动作 或 状态是 **尚未发生** 或者 **即将发生** 的

含义: **在做A部分之前, 先将B部分做了**

```s
# 去日本的时候, 买了很多礼品 (礼品在中国买的), 先买的东西, 去日本的事情还没有发生呢
日本へ行く時、たくさんお土産を買いました

# 出门的时候，请关灯
出かける時、電気を消してください

# 在图书馆看书的时候，戴眼镜
図書館で本を読む時、眼鏡をかけます

# 看电影的时候, 总是坐在后面, 这里是先坐下再看的含义, 后项的动作完成后 电影还没有开始
映画をみるとき、いつも一番後ろの席に座ります
```

<br>

### 动词た型 + 時、〜
当 动词た型 + 時, 表示 **前项的动作已经完成后, 后项才完成**, 也就是说, **A部分的动作或状态是已经发生的**, 主句的动作发生在从句的动作之后

含义: **A部分之后, B怎么样了**

```s
# 去日本的时候, 买了很多礼品 (礼品在日本买的), 去日本之后, 买了很多礼品
日本へ行った時、たくさんお土産を買いました

# 旅行结束的时候，拍了很多照片
旅行が終わった時、たくさんの写真を撮りました
```

<br>

### 体言 / 形1 / 形2 + 時、〜
表示 前后两项的状态 **同时进行**

```s
# 两个状态是同时发生的
子供の時、大きな地震がありました

# 正在忙的时候 来电话了
忙しい時、電話があったんです
```

<br><br>

## 动词连用形 + ながら、〜
表示**前后两项的动作是同时进行**, 一般以**后项的动作为主动作**, 翻译: **一边...做什么, 一边...做什么**

同一主体, 同一时间, 一边... 一边(主动作)...

**回顾: たり、たり、する**  
1. 同一主体 不同时间 有时...有时...
2. 不同主体 同一时间 并存, 既...又...

```s
# 一边打电话, 一边开车是很危险的, 同时这个句子中开车是主动作
携帯電話をかけながら、運転するのは危ないです

音楽を聴きながら、宿題をしてます
```

# 意志形

```s
# と思ってる 一直在想
来年、留学しようと思ってる
```

### 动词意志形 + と思う 和 たい 的区别
- やりたい
- やろうと思う

上面的两种说法都是想表达我想要做 区别在于

<br>

**たい:**  
表示的是 单纯的意愿, 愿望, 内心对某行动的希望和向往, **即使不能实现也没有关系**

<br>

**动词意志形 + と思う:**  
表示的是 想要做的事, **有了初步的想法和打算 之后会去做**

```s
休もうと思う # 想要休息, 一定会去请假
```

<br>

### 应用2: 动词意志形 + とする
1. 正打算做某事, 正要怎么样
```s
# 出ようとした 已经决定好了要出门
家を出ようとした時、電話が鳴りました

# 正打算做
今宿題をやろうとしてる
```

2. 某种行为 或 某件事情 正在开始 或 正处于进行前的状态 (正要干的事情处于进行前的状态)
```s
    正要做的事, 但是正处于 . 位置的状态
    ↓
---.--------

# 含苞欲放的感觉 但是还没开
桜の花が今、咲こうとしています

# 正要发车, 但是还没有走的时候 发生了什么
電車が発車しようとした時、地震がありました
```

<br>

### AをBとする
- 用法1: 它前面可以加 意志形 表示 正要做什么, 正打算做什么

- 用法2: 把...作为 / 当做, 将本来不是B的A, 一时作为B来用, **强调是暂时的, 表面的变化**

<br>

**AをBとする 和 AをBにする 的区别:**  
1. とする表达更加的书面一些
2. にする是**将A长期的, 永久的当做B, 是一种本质的变化**
3. とする是将A暂时的当做B

```s
# 将那个房间暂时当做仓库
その部屋を　倉庫としました。
# 将那个房间改造成仓库
その部屋を　倉庫にしました。

# として是て断句, 暂时的将図書館当做教室用
図書館を　教室として、来週から一ヶ月勉強をします
```

<br>

### V简体 / 形容词 / Nな + ので、〜
表示因果关系, 前句为客观原因, 后句一般不接推测, 意志, 命令等语气, 其口语形式为 ``んで``

- ので: 前面接的是 客观原因
- から: 前面接的是 主观原因

```s
荷物が重いので、宅配便で送ります

ここは静かなので、とてもきに入りました

もう終電の時間なので、帰ります

昨日雨が降ったので、出かけなかった


# 主观的原因: 主观认为便宜
安いから、買いました
# 就是便宜
安いので、買いました
```

<br>

**注意:**  
1. 小句2的部分, 表示 **命令, 禁止, 提议, 建议** 的时候 必须使用 から

```s
# 已经没有时间了, 快走吧
もう時間がないから、早く行こうよ ✅

もう時間がないので、早く行こうよ ❎

危ないから、触るな
```

2. 表示 **请求**, **道歉** 麻烦别人等的时候使用 ので 更加的合适 因为是客观的原因
```s
今忙しいので、この荷物、お願いします

電車が遅れたので、遅刻しました、すみません
```

# 条件句(动词基本形(原型)/ない形)と、结果句
〜と、用在表述状态, 真理, 反复性状态, 习惯等内容中, 表示前面是后句的条件

另外, 它还可以表示, 由于某种行为而发现了新的情况
```s
この道をまっずく行くと、デパートがあります
```

<br>

### 翻译: 
必然的结果, 只要符合前面的条件 结果就会发生, 它是逻辑上或时间上的必然关系

1. **只要...就... / 如果...就...**, 必然的结果(真理, 规律, 现象等), 是一种逻辑上必然的关系

2. **一...就...**, 时间上, 一到春天花就开了

3. ない形と: **没有...就...**

<br>

### 场景:
1. 恒常性的 自然性的 状态
  - 一到了春天花就开了
  - 一打雷就下雨
  - 一插电源, 就有电了
```s
ここに立つと、ドアが開きます
```

2. 真理
  - 如果人类没有水的话就无法生存
```s
水がないと、人間が生きることができません
```

3. 反复性的动作
  - 一去麦当劳就会点个甜筒
  - 一去万达影院就会在最后一排
```s
マクドナルドに行くと、
いつもアイスを注文します
```

<br>

### 注意:
结果句中 **只能是客观的陈述**

不能是自己的意志, 愿望 或向 听话人提出要求, 劝诱等内容
```s
夜になると、食事に行きましょう ❎

夜になると、電気がつきます (到了晚上灯就亮了) ✅
```

<br>

```s
春になると、はなが咲きます

気温が低いと、さららが咲きません
```

- 他动词: 由他人的力量导致了动

## 自动他动的区分:

### 第一种: 以 自然 和 人为 来进行区分
都是 人为 的时候 我们看 是自主的, 还是外界力量推动的

<br>

### 第二种: 按助词来区分
- 自动词: が
- 他动词: を

<br>

## 自他动词的小规律
我们以日文的罗马发音为准, 有一些如下的规律, 下面的两种规律只占了30%

<br>

**规律1:**  
1. 自动词词尾部分是 ``aru``
2. 他动词词尾部分是 ``eru``

比如
```s
掛かる　kakaru -> aru
掛ける　kakeru -> eru
```

<br>

**规律2:**  
1. 自动词词尾部分是 ``eru`` 或者 ``ru``
2. 他动词词尾部分是 ``su``

比如
```s
消える　kieru -> eru
消す　　kesu -> su
```

<br>

## 自动词:
不需要宾语的动词 (不及物动词)

自动词**它强调物体 本身自己的状态**, 物体本身在动, 不需要外力, 不强调动作的执行者, 甚至没有动作执行至, 单纯表达状态

提示主语使用 **が**

比如: 自动门, 门自己在动 这就是自动词, 而我们手推门则是他动词

例如:
```s
働きます　ねます　おきます

# 手紙 是 主语, 当前面的主语成为整个句子的主题的时候 则可以将 が 换成 は
手紙が届きます
```

<br>

### 自动词 + ています
以前我们学过很多 ``ている`` 的使用方式 比如
1. 正在进行
2. 结果状态
3. 反复, 习惯
4. 职业 

<br>

这里我们是 ``自动词ている`` 表示 自动词这个动作的 结果的状态 或 结果的存续

- 如果该自动词表示动作的, 则``自动词ている``表示动作正在进行
- 如果该自动词表示变化的, 则``自动词ている``表示结果的存续

```s
# 房间的灯 灭着
部屋の電気が　消えています

# 墙上挂着一张大画: 强调的是画挂在墙上的这种状态
壁に大きな絵が　掛かっています

# 这辆车坏了
この自動車が　壊れています

# 强调剪刀放进抽屉里之后 就一直在抽屉里
ハサミはどこですか
引き出しに入っていますよ
```

**自动词ている and 自动词た形:**  
这两个语法都可以表示结果的存续, 但是 ``自动词た形`` 偏重表示变化的完成, 而``自动词ている``则偏重表示变化完成后形成的结果状态

```s
# 房间的灯 灭着: 强调的是 没电了后的状态
部屋の電気が　消えています

# 房间的灯灭了: 瞬间的变化
部屋の電気が　消えました
```

<br>

### Vて形しまう (てしまう === ちゃう / でしまう === じゃう)
该语法有两种含义
1. **表示做完某事后**, 该动作所产生的结果有点 令人不愉快 或 遗憾等情绪 (比如辛辛苦苦大半年一下回到解放前), **就是说可以理解为和 动词た形 的使用方式一样, 但有其他的情绪在里面**
```s
# 森先生把奖金全部都花光了
森さんはボーナスを全部　使ってしまいました
```

2. 单纯的强调事情做完了, 没有其他的情绪
```s
# 这本书已经全部读完了
この本を　もう読んでしまいました
```

<br>

### 动词ます形 / 形容词 + そうです
**翻译:**  
看起来...怎么样 或者 好像...怎么样

用于描述说话人对某种事物的**观感** 或 **根据某种情形**, **推测**事态的发展

<br>

**接续:**  
1. 动词去掉ます 接 そうです
```s

```

2. い形容词 去掉い  接 そうです
```s

```

3. な形容词 直接接 そうです
```s
大変そうです
```

4. ない 和 いい 的时候 い -> さ　接 そうです
```s

```

<br>

**否定的表现形式:**  
- 前接 形容词 的否定形式: そうではない 比如 ``美味しそうではありません``

- 前接 动词 的否定形式: そうにない 比如 ``雨はやみそうにありません``

<br>

**そう作为小句接续:**  
我们将そう看做为2类形容词

- 接名词: そうな
- 接动词: そうに

```s

```

## 他动词:
需要宾语的动词 (及物动词), 通过其它的力量让某一个物体动起来 是外力的作用

受到外力影响, **该动作 作用在宾语上**

提示主语使用 **を**

例如: 类似下面的动词, 只有动词的话是不完整的, 我们要表达吃什么 喝什么 看什么 这类动词就是需要宾语的动词
```s
食べます　飲みます　みます

ドアを開けました
```

<br><br>

## 他动词 + てあります
该句型和 ``自动词 + ています`` 一样 都是表达 物体的状态, 比如下面的例句中 我们都可以翻译为 画挂在墙上

但是 具体还是有区别的

``自动词 + ています`` 该句型使人感觉不到 动作者的存在, 看到的只有画, 看不到人, 只是强调物体的状态

``他动词 + てあります`` 该句型 **动作者本身不出现在句型中 但是能感觉到动作者的存在**

```s
# 掛かる: 自动词 的 挂: 自动词ています 表示 画一直挂着, 强调的是 我们进到屋里的时候 画它自己就在那挂着, 不知道谁挂的, 我也不在意是谁挂的, 我只是想告诉你画挂着, 只是单纯的强调状态
絵は掛かっています

# 掛ける: 他动词 的 挂: 画一直挂着, 强调的是 人为, 是有一个人将画挂在墙上, 画被某人挂在了墙上 之后是画挂着的一个状态
絵は掛けてあります
```

<br>

### 区别:
它们都是表示 物体的状态, 我们可以这么想
- 如果是 自动词 的话我们就使用 ``自动词 + ています``, 它强调的是物体本身的状态

- 如果是 他动词 的话我们就使用 ``他动词 + てあります``, 它强调的是 人为, 是有一个人将画挂在了墙上, **强调的是有人刻意而为之的一个动作**

它们之前强调的侧重面不一样, **但是从结论和翻译的角度最终是一样的**

<br>

### 例句:
```s
# 窗户开着
窓が　開けてあります

# 护照在包里放着
パスポートは　バッグの中に　入れてあります
```

该句型表示有意进行的动作结果的存续状态, 什么叫做有意进行的动作和其结果状态

```s
# 我在墙上挂了日历
壁に　私が　カレンダーをかけました

# 墙上挂着日历
壁に　カレンダーが　かけてあります
```

<br>

### 注意: 助词的问题
原则上 ``N を 他动词`` 他动词的宾语使用的是 ``を``, 
他动词的作用的对象, 我们使用 を 来提示

``他动词 + てあります`` 表示的是持续性的**状态**, 既然是状态, 那就是一直持续下去的东西, 所以动词前面的助词使用的是 **が**

**使用 が 来提示状态的主体**

```s
# 在桌子上 装饰了花
机の上に　花を　飾りました
```

但是我们要使用 ``他动词 + てあります``　表示物体的某动作之后的状态的时候, 我们要使用助词 ``が``, 表示装饰完的话 放在那是一直存在的状态

```s
# 桌子上装饰着花
机の上に　花が　飾ってあります

# 花装饰在桌子上
花は　机の上に　飾ってあります
```

所以我们有如下的两个句型哦
- 〜に〜が　Vてあります
- 〜は〜に　Vてあります

2. 特殊点的用法 ``场所でも``, **翻译: 在什么地方也...**
```s
# 成龙
ジャッキ・チェンは日本でも有名です
```

## Nだけ
表示这就是全部, 不再有其它的意思, **翻译: 只, 仅**
```s
# 今年的暑假只有3天
今年の夏休みは三日だけです

# 只有xxx喝酒
森さんと馬さんだけが、お酒を飲みます

# だけ后面的助词 が 或者 を 有时会省略
私は その料理だけを 食べませんでした
```

<br>

### 助词位于 だけ 前后均可, 但是 で 前后语感有差别
- だけで: 这里的で表示手段方法, 一般用于, **只需要...就可以把事情办好, 无须其它的办法**
```s
バスだけで、来る
```

- でだけ: 这里的で表示手段方法, 一般用于, **只有靠...才能办好事情, 其它的方法都不行**
```s
# 只有做公交车 才能来
バスでだけ、来る
```

<br><br>

## Nしか+否定形式
表示除了列举出来的这项外, 其它的都不是这样的, 类似 ``〜だけ+肯定形式``

**翻译: 只, 仅 或者 除了前面的名词 剩下的都不...**

```s
# 会议室只有小李一个人
会議室には、李さんしかいないです
会議室には、李さんだけいます

私は水しか飲みません
私は水だけ飲みます
```

<br>

### 区别
两者的区别在于``〜だけ``只用来表示一种客观的限定, 而``〜しか〜ない``带有主观的感情色彩

``〜だけ``和``〜しか〜ない``在接数量词的时候, 语感有强烈的不同
```s
# 主观上觉得3天时间太短了
北京には三日しか滞在することができません
# 客观上表达只在北京停留3天
北京には三日だけ滞在することができます
```

<br>

### 注意: 助词部分
助词 は　が　を　可以省略, 其它助词(に、へ)位于しか前面
```s
# 只对你说了
君にしか話しません
```

<br>

### 注意: 否定句的限定 只能用 だけ
只有我不知道
```s
私だけ知らない
```

## 动词简体形・Nの + とおりに
日语中的常用固定性的一种说法, 表示 **按照...那样...**

通り 本身是 马路的意识, 引申义为 沿着, 顺着, 所以我们这个句型就可以翻译为 按照...那样...

```s
# 如天气预报说的那样, 下大雨了
昨日は天気予報のとおりに、大雨が降りました

教えた通りにやってみてください

思った通りのものでした
```

## N に (小句) / 动词基本型の に (小句)
动词小句加 に 的时候, 需要将前面的小句名词化 也就是 ``基本形+の``

这两个句型都是用来表示 用途 和 基准

**翻译:**  
- 用途: 用来做什么用的 (申請に使います)
- 基准: **对于xxx来说, 怎么怎么样** (子供には難しい / 買い物に便利)

<br>

### N に 小句:
1. 表示 用途 时, 前面(N的部分)是具体说明用途的名词, 后面一般是 ``使います`` 等动词
```s
# 这张照片做什么用
この写真は何に　使いますか

# 申请护照的时候用
パスポートの申請に使います
```

<br>

2. 表示 基准 时, 前面(N的部分)是名词的基准, **后面一般是表示评价的形容词**
```s
# 离超市很近, 这个公寓出去购物非常方便
# 買い物: 买东西是基准点
# 便利: 对于基准点的评价
ズーパーが近いので、このマンションは買い物に便利です

# 这本书对大人来说很容易, 可以对孩子来说太难了
この本は、大人には易しいですが
しかし、子供には難しい
```

<br>

### 动词基本型の に (小句)
这个句型的作用和上面一样

```s
# 这张照片申请护照时用
この写真はパスポートを申請するのに使います

# 这个平底锅是煎鸡蛋用的
このフライパンは卵焼きを作るのに使います

# 这个公寓出去购物非常方便
このマンションは買い物を買うのに便利です

# 为了适应北京的生活 是不是做了很多
北京の生活に慣れるのに　苦労しましたか
```

## N ばかり V动词 / 动词て形ばかりいます
这两个句型都可以 翻译: **尽... / 光... / 一味地... / 总是...**

<br>

### N ばかり 动词
表示所列举的事物全部相同, 

```s
# 为什么光吃蔬菜呀?
なんで、野菜ばかり食べてるんですか？

# 林先生唱卡拉OK尽唱老歌
林さんはから、古い歌ばかり　歌います
```

<br>

### 动词て形ばかりいます
表示总是发生同样的事情 或 总是进行同样的动作

```s
# 小长整天光喝酒
長さんは毎日　お酒を飲んでばかりいます

# 森先生老是出岔子
森さんはいつも失敗してばかりいます

主人は働いてばかりで、どこへも行かないですよ
```

<br>

### だけ 和 ばかり 的区别:
ばかり不能做的, だけ都能做

ばかり 和 だけ 都可以表示限定, 但其用法有以下3点不通
1. だけ可以用来限定数字, 而ばかり不行
```s
３つだけ食べました
```

2. だけ可以和否定形式连用, 而ばかり不行
```s
その料理だけを食べませんでした
```

3. だけ只表示没有例外, 而ばかり则含有反复进行同一动作的意思
```s
# 只喝了一杯茶, 没有喝其他的东西
お茶だけ飲んでる

# 一个劲的喝茶, 一杯又一杯的喝茶
お茶ばかり飲んでる
```

4. **ばかり有一种不太理想的含义**

## 可能形
有些教材中会将可能形用在能力上, 比如 我能走, 能说, 能跳舞, 但它偏向于 客观性, 自发性, 不带有意志性的表达 (非意志性)

<br>

### 场景
比如 我能说日语 这句表达, 它不是炫耀我能说日语可厉害了, 而仅仅是表达的客观情况, 单纯的表达这种客观情况和能力

1. 能力: 比如能吃 (有的人花生过敏, 客观的表达是否能吃)
2. 许可: 我能吃个糖么?

可能形的主观情绪比较少, 比如 ``使えるか`` 在使用在问句中的时候, 它想表达的不是老子能用, 而是想表达我能使用么? 是请求许可, 决定我们能不能用, 可以不可以使用的对象不是我们自己 是别人

和我们之前学的 ``〜ことができる`` 几乎是一模一样, 但使用可能形更加的容易 方便

1. ``〜ことができる``: 也有比较 客观 正式 的感觉
2. ``可能形``: 比较主观

<br>

### 变形
1. 五段动词: う段 -> え段 + る: 書く -> 書ける
2. 一段动词: 去る + られる: 食べられる
3. する -> できる 和 来る -> 来られる

<br>

### 可能形的助词
可能形动词前面的助词不是 を 而是 **が**, 因为可能形带有客观性 所以前面的助词使用 が

或者 我们也可以这么记, 在日语中表示情感能力的动词 前面的助词使用 が, **可能形都属于自动词, 所以用が**

<br>

### 扩展:
日本年轻人追求的是说话 又短又快 下面的说话从语法角度来说 是错的, 但很多人在用 就是将 一段动词中的 ら 省略掉

```s
食べられる ->　食べれる
```

<br>

### 小句(原形 / ない形) + ように、小句 
小句1中的动词多为表能力的词汇, 一般前面接
1. 非意志性动词的 原形 / ない形
2. 意志动词的 可能形 / ない形

<br>

**非意志性动词:**  
指那些不涉及主语意志或意愿的动词

这类动词表示自然发生的事情、状态或能力，通常不带有主语的主观意图

- ある（存在，有）
- いる（存在，有）
- できる（能够）
- 見える（看得见）
- 聞こえる（听得见）
- 起こる（发生）

<br>

**意志动词:**  
涉及主语的意图或意愿的动词

- 食べる（吃）
- 行く（去）
- 読む（读）
- 書く（写）
- 話す（说）
- 遊ぶ（玩）

也就是说 ように 前面是不能接主观的表达的 我自己决定的表达是不能接的 但是一个意志动词变成可能形之后就可以接了, 因为可能形式是自动的, 非意志的 客观的

我们可以理解为 只要 ように 前面的表达没有自主的意志 没有自己的想法 不是我们当机立断就能做的事 都可以直接接这个语法

<br>

**翻译:**  
...为了能..., 而...

...为了不能..., 而 / 以免 ...

```s
# 自动词原形 直接加
よく　見えるように、大きく書きました

# 感冒不是自己能控制的 所以直接接
風邪をひかないように、気をつけています

# 做好了随时能够触发的准备
# 出る 是意志动词 我们自己能决定是否出去的 所以变成可能形
いつでも、出られるように、準備してあります

# 这个药放在橱柜的最上层吧, 免得孩子动他
子供が　触れないように、この薬は棚の一番上に置きましょう
```

<br>

**ように 和 ために 的区别:**  
两者翻译都是为了, 但是用法不同, 不能互相替换

ように前面一般都是 非意志性的变化或状态, 因此 ように 往往前接可能形式的 原形 或 ない形

而 ために 表示的意志性动作的目的 (**前面接意志形的**), 因此通常前接动词的原形

```s
日本に　留学するために、貯金しています

日本に　留学できるように、貯金しています
```

另外 ように 前面的主语可以与后面句子的主语 不一致, 但是 ために 则不行 它的前后主语必须一致

```s
他の人が怪我をしないように、森さんは割れたコップを捨てました

森さんはコンサートのチケットをかうために、並びました
```

<br>

### 小句(原形 / ない形) + ようになります
表示 能力, 状况, 习惯 等变成了某种状态 常态

小句1中的动词多为表能力的词汇, 一般前面接 (解释看上面的句型)
1. 非意志性动词的原形 / ない形
2. 意志动词的可能形 / ない形

```s
# 伤已经痊愈了(自然而然好的) 能够走路了
怪我が治って、歩けるようになりました

# 已经不会搞错平假名了
カタカナを間違えないようになりました

# 已经能够自己穿和服了
着物を自分で着られるようになりました
```

<br>

### 小句(原形 / ない形) + ようにします
表示 努力 使某种行为, 状况变成现实

当表示平时留心做某事已经成为习惯的行为时, 常常使用 ``ようにしています``

**翻译:**  
1. ようにします: 翻译为 尽量努力做到
2. ようにしています: 翻译为 坚持...
3. ようにしてください: 翻译为 希望 务必 尽量

```s
# 老陈坚持每天看英文报纸 (想将 新聞を読む 变成一种常态, 通过自己的努力和意志)
陳さんは毎日新聞を読むようにしてます

# 每天早上坚持7点起床
毎朝、7じに起きるようにしてます

# 因为嗓子疼, 今天要少说话
喉が痛いので、今日はあまり話さないようにします

# 为了健康, 不熬通宵了
健康のために、徹夜しないようにします
```

<br>

### 小句(原形 / ない形) + ようにしてください
翻译: 尽量 / 记得 / 务必

```s
写真は撮らないようにしてください
```

## V + ところです

### ところ本意
1. 地方, 地点; 住处
2. 点: 彼にはいいところがある
3. 时间 / 期间: 今のところ、大丈夫？

<br>

我们今天的句型 就是在期间 和 时间上的一些扩展

<br>

### 接续:
1. 动词基本形(原形)
2. 动词ている
3. 动词た形

<br>

### 翻译:
表示 某事 或 某动作 正处于...阶段 (**时态不同处于的阶段也不同**)

1. 动词原形: 将要开始做... / 某动作即将开始 (正要...)
2. 动词ている: 正在进行中
3. 动词た形: 结束的阶段

不单纯的描述某个进展阶段, **多有上下文**

<br>

### 场景1:
该句型的目的是用来说明自己的一些状况的

比如 对方问你能出差么? 我们在进行说明的时候 使用该句型

也就是说, **应答**对应的提问, 邀请等, 目的在于**说明状况**, 多用于告知, 解释, 辩解, 拒绝等

<br>

### 场景2:
说明自己的状况, 提出建议, 劝诱等

比如 我正要去吃午饭呢, 要不要一起去呀, 也就是我们处于将要去食堂吃饭的阶段下, 我们对对方进行邀请

<br>

### 动词原形 + ところです
将要开始做... / 某动作即将开始 (正要...)

```s
# 我正要和朋友一起去吃饭, 一起不?
# 场景2的使用方式
これから、友達と食事にいくところです
　一緒に行きませんか？

# 我正要出门
これから、家を出るところです
これから、家を出るところで、20分ほど着くと思います

# 我现在正要洗澡, 待会给你去电话
今、風呂に入るところなので、後でこちらから電話します
```

<br>

### 动词ている + ところです
表示动作的持续, 正处于在做某件事的阶段 (正在...)

```s
# 正处在正在调查的阶段
今調べてるところですので、少しお待ちください

# 森正在整理会议资料
森さんは会議の資料を揃えてるところです

# 我正在看书, 等下再回去
ちょうど本を読んでるところなので、後で帰ります
```

<br>

**注意:**  
该句型中的动词不能是 非意志性动作 和 瞬间动词
```s
# 错误示例: ❎
今、ドアが開いてるところです

# ❎
雨が降っているところです

# ❎ 瞬间动词
曇りが消えているところです
``` 

<br>

**补充: Vていたところです**  
一直在做某事持续到现在

```s
# 我刚才一直想着给你打电话呢, 没想到你的电话就来了
私もちょうどあなたに
  電話しようと思っていたところです

ケーキが食べたいと思っていたところなの
```

<br>

**ている vs ているところだ:**  
``ている`` 单纯描述某个动作正在进行  
```s
# 单纯的叙述
あの人は故障の原因を調べています
```

``ているところだ`` 意在说明正在忙于手头的事情 **会有前后文**
```s
原因が分かりましたか？
いえ、今調べているところです、分かったら、すぐ連絡します
```

``ている`` 可以做定语 来修饰体言

``ているところだ`` 无法做定语

<br>

### 动词た形 + ところです
表示动作或事件 **刚刚结束**, 该动作正处在刚结束的状态 (刚刚)

```s
# 刚从学校出来 (从踏出学校的大门)
今学校を出たところだ
  これからバイト先に行く
```

长于 今、たった今、ちょうど 等副词一起使用

<br><br>

## V + たばかりです
表示 动作 或 事情 **刚刚结束不久**, 与 Vたところです用法基本相同

```s
このモノレールは、去年開通したばかりです

先月、中国に来たばかりですから、
  中国語がまだ下手です
```

<br>

### 注意:
``たばかりです`` 可以表示动作行为结束后经过了较长一段时间的事态 是**我们自己主观觉得的刚刚**

但是　

``た形 + ところです``则不行 它表示动作行为结束后经过的时间极短就是刚刚, **动作刚完成的时间点**

<br><br>

## 动词连用形 + 始める・出す
都是出现和开始, 但是出す强调的是不可控 突然

- 动词连用形 + 始める: 表示该动作的开始
- 动词连用形 + 出す: 出す是从里往外的出, 比如可以说 ...出来了, ...起来了, 表示动作的出现 (**偏重突然, 不可控, 意外的出现**), 注意它不用于``しましょう``和``意志句``中

```s
# 开始下雨了; 下雨的动作开始了
雨が降り始めました

# 渐渐的人们开始回去了
段々人が帰り始めたね

# 这本书昨天刚刚开始读
この本は昨日読み始めたばかりです

# 突然开始下雨 (在表示突然性的时候我们会选择使用 出す)
急に雨が降り出した

# 孩子突然哭起来了
子供が泣き出して、困ります

# 突然间生气起来, 吓了一跳
急に怒り出して、びっくりした

# 突然笑出声了
笑い出してしまった
```

<br>

**惯用句:**  
- 思い出す: 记忆中想起某件事
- 作り出す: 做出来了
- 考え出す: 思考出来了

<br><br>

## 动词连用形 + 続ける
表示某动作或状态 **不间断的持续**

```s
３時間　歩き続けました

車のオイルが漏れ続けてます

# 从两个小时前起一直站着, 站类了
２時間前からずっと、立ち続けています。 
  もう疲れました
```

<br>

### 注意:
降る 不能接 続ける, 而是接 続く
```s
雨が１週間　降り続きました
```

<br><br>

## 动词连用形 + 終わる
表示持续性动作 或 行为的**结束**, 用于该动作, 行为具有一定的量, 而且行将结束

```s
# 把那本书看完了
その本はもう、読み終わりました

# 读完后
読み終わったら、その本を返してください

# 上周拿的药已经吃完了
先週もらった薬は　もう飲み終わりました
```

## 被动型

### 介绍
在最早日本没有文字, 甚至没有形成语言体系的时候 讲话都是很简单的 **主谓宾** 关系, 日语语法中会使用 **主宾谓**
```s
主      宾      谓

我      饭      吃

私は　  ご飯を　食べる
```

这个时候人们交流是非常基础的
- 我在干什么
- 你在干什么
- 我想干什么
- 你想干什么

主谓宾在一句话中都非常的清晰, 也就是说最早的时候, 语言体系中只有简单句, 但是后来发现这种简单句无法表达很多的内容 会产生一个问题 就是主语不见了

主语在日语中会使用 ``は・が`` 来提示

```s
宾      谓

饭      吃

ご飯　  食べる
```

主语部分不见了 ``は・が`` 没有用了

饭不知道被谁吃了 - ``饭 被 吃`` / ``水 被 喝``, 在这样的句子中 **饭 和 水 成了我们想要强调的重点**

因为 饭是被谁吃的? 水是被谁喝的?

所以日语中出现了一种情况 就是使用 ``は・が`` 这种用来提示主语的助词 放到了 饭 和 水 这样被动型的主体当中

从语法的根源来讲这种做法是很诡异的 因为吃的是饭, 喝的是水 应该使用助词を 来提示动作的对象

结果却是 日本产生了一套表达体系中表示
- 被吃的
- 被喝的
- 被玩的
- 被用的 东西

这个东西, 这个主体 我们会使用 ``が`` 来提示, **作为被动态的主体**

```s
# 小李被表扬
李さん　は・が　褒められた

# 被谁, 使用助词に提示
李さんは　xxに　褒められた

# 表扬了什么?　表扬的具体的内容使用を
李さんは　xxに　yyを　褒められた

# 如果没有具体的内容 yyを 可以省略
```

<br>

### 变形
1. 五段: う段 -> あ段 + れる -> ``使う -> 使われる`` / ``書く -> 書かれる``
2. 一段: 去掉る + られる -> ``食べられる``
3. くる -> ``こられる``(被谁要求来) / する -> ``される``

变形后 都相当于 一段动词 因为以れる结尾 否定 过去 都按照一段动词变

<br>

### 直接被动句
1. 主语是动作行为的直接接受者, 动作的行为 **直接** 涉及 **接受者本身**
```s
# 语法
主语は／が、动作实施者に　他动词＋(ら)れる。

王さんは　犬に　噛まれた。(小王被狗咬了。)
```

2. 主语是 动作行为的直接接受者, 行为 **直接** 涉及 **接受者的某部分或所有物**
```s
# 语法
主语は／が、动作实施者に　所有物を　他动词＋(ら)れる

王さんは　犬に　かばんを　噛まれた
```

<br>

### 间接被动句
动作行为 **不直接** 涉及被动句的主语, 但是给主语带来了麻烦或损害

```s
# 语法: 自动词
主语は／が、动作实施者に　自动词＋(ら)れる。

せっかくの休みなのに、雨に降られて、どこへも行けなかった


# 语法: 他动词
主语は／が、动作实施者に　动作对象を　他动词＋(ら)れる。

私はレストランで周りの人にタバコを吸われると、気分が悪くなる。
```

<br>

### 自然表明句
客观地描述某种时态和状况, 动作主体为某个不确定的人或群体

有的时候**不是特别注重行为者本身**, 而把事情或者物品作为主题 这时使用被动时态

```s
主语(动作对象)は／が、 他动词＋(ら)れる。  

2021年のオリンピックは、東京で開かれました。
```

<br>

### 补充: 动作施动者后用除了「に」, 还有以下助词的使用
|谓语动词|助词选择|
|:--|:--|
|传递意义动词 送る、届ける、渡す、与える、伝える　など|から|
|创作、发明、建造等意义的动词 作る、書く、発明する、建てる、設計する　など|によって|

<br>

### 句型: 名词1は 名词2に 动词られる (直接被动句)
1. 动词作用在 名词1 上 (名词1是动作行为的直接接受者)
2. 发出动作的人是 名词2 (名词2是动作的实施者)

```s
李さんは　部長に　褒められた
```

**注意:**  
表示受益时通常使用 てもらう 的形式

```s
# 小李得到了朋友的帮助 - 客观叙述, 不表示受益
李さんは　友達に　助けられた

# 小李得到了朋友的帮助 - 表示受益
李さんは　友達に　助けてもらった
```

<br>

### 句型: 名词1は 名词2に 事物(东西)を　动词られる (直接被动句)
1. 名词1 是 动作的直接接收者
2. 名词2 是 动作的实施者
3. 事物是名词1的

动作行为 **不直接** 涉及到名词1(主语), 但是给主语带来了麻烦或损害

```s
# 小李的包 被狗咬了 李さんのカバン
李さんは　犬に　カバンを噛まれた

# 马先生的照相机被森先生弄坏了
# 小马让森先生把照相机弄坏了 好奇怪的翻译
# 1. 看主语: 马先生
# 2. 看动词: 被搞坏了照相机
# 3. 看に: 被谁 被森
馬さんは　森さんに　カメラを壊されました

# 小李昨天被狗咬手了
李さんは昨日　犬に　手を噛まれた
```

<br>

### 句型: 名词1は 名词2に 动词られる (间接被动句)
1. 名词2 是动作的实施者
2. 名词2に 动词られる 这个部分 **对 名词1(主语) 带来了麻烦或损害**

也就是 ``名词2に 动词られる`` 这个部分的动作 **虽然没有作用在主语上, 但是这个动作给主语带来了损害**

```s
# 陈的小鸟飞走了 陈样了多年的小鸟
陳さんは　小鳥に　逃げられた

# 1. 先看 后面的部分: 邻居闹到很晚
# 2. 長さん 被影响到了
長さんは　隣の人に夜遅くまで　騒がれました

# 1. 先看 后面的部分: 旁边有人抽烟
# 2. 我被影响到了
食事中、隣の人にタバコを吸われました
```

**注意:**  
这个句型中 名词2 的部分一般都是 人或动物, 但有时候也可以是 雪 或 雨

```s
# 我们被雨淋湿了
私たちは　雨に降られました
```

<br>

### 句型: 名词は・が 动词られる (自然表明句)
用事物来作为主语的被动句

1. は: 提示 后项 内容
1. が: 提示 前项 内容

客观地描述某种时态和状况, 动作主体为某个不确定的人或群体

有的时候**不是特别注重行为者本身**, 而把事情或者物品作为主体 这时使用被动时态

```s
# 万博会被召开, 我们不知道是被谁召开的 也就是 はが提示主语不知道提示说, 而我们关注主语, 而把事情或者物品作为主体 这时使用被动时态
2010年に上海で　万博が開かれる

# 车站附近盖起了高楼
駅前に　高いビルが　建てられた
```

或者记 人类行为产生的大型活动 就可以像上面这样表达

<br>

### 句型: 名词1は 名词2によって 动词られる
``名词2によって`` 这个部分和上面我们使用到的 ``名词2に`` 的使用方式是一模一样的

这里的``によって``表示强调, 当名词2是比较有名望 有声望 大众的 厉害的 权威的 这样的身份是 我们会使用 ``によって``表示强调

```s
# 这款车是由日本著名设计师设计的
この車は　日本の有名なデザイナーによって、設計されました
```

**名词2的部分很牛逼的时候使用 によって**

<br>

### 句型: 动词られる
被动句的用法 一般在日语**书面语**中 常常用被动句 表示 客观 和 礼貌

这样会让整个报告书 看起来是客观的事实 真想, 而不是谁的主观想法, 通过被动来表达是客观的事实 (抛开个人情感)
```s
低価格が実現できたと考えられた
```

## 使役被动形
当某人**被迫或不由自主地**做某件事情时使用使役被动。

### 变形:
其实使役被动是动词
- 先变成使役态后
- 再变成被动的形态。

<br>

1. 1类动词: い段 -> あ段　＋　されます
  - い -> わ: 買います -> 買わされます
  - し -> さ: 話します -> 話させられます
```s
飲みます　飲まされます
```

2. 2类动词: ます -> させられます
3. します -> させられます / 来ます -> こさせられます

<br>

### （迫于形势）A被B强迫做（不情愿做的）事时候使用。
```s
# 语法: 
主语(承受者)は／が、(誰か・某人)に動使役被动形

子供は毎日母に野菜を食べさせられる。
孩子每天都被妈妈要求吃蔬菜。
```

### 由"自发态＋使役形"构成, 表示使人不由自主地产生某种心理或生理活动。
```s
その話を聞いて、私は大いに考えさせられました。／听到那些话, 让我不由得想到很多。

彼の言ったことについては本当に考えさせられる。／关于他说的事情, 不得不使我认真思考一下。
```
