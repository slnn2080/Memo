## 常用副词

### やっぱり・やはり
1. 表示事物的发展正如自己预料, 翻译: 果然
```s
やっぱり　あめがふりましたね
```

2. 经过一番思考确定某想法, 在一堆事物中选定某个, 翻译: ``想来想去, 还是...``, ``说来说去, 还是...``
```s
どのお茶が一番　人気がありますか
やっぱり　烏龍茶です
```

<br>

### 最近　さいきん
最近, 近来

在日文中, 最近 只能表示由较近的过去到现在为止的一段时间, 而不能表示由现在到将来的某一种时间

也就是说它表达的是过去时态

最近并不是对时间的长短有限制, 而是根据说话人的语境决定的

<br>

### ところで
转换话题 一般不用翻译, 如果硬要翻译 相当于 诶? 说起来  
```s
きょうは暑いですね

ところで、今朝のニュース見ましたか
```

<br>

- あまり: 不太

<br>

### まっすぐ

<br>

### なかなか
很难 不容易、相当 很 非常

后接肯定的时候, 表示: 相当, 很, 非常 (暗含比我想象的要好嘛的意思)

なかなか这时的语气是说话人位于上层, 带有评价的口吻, 因此不适用于谈及上司和长辈的情况

```s
# 表示时态的发展并没有所期待的那么简单 或者 不能轻易实现
# 我照着书做也老是做不好
本の通りに作っても、なかなかうまくできない
```

<br>

### 結構です
1. 表示程度比自己预想的要高

和上面的 なかなか 意思相近, 但是 なかなか 只能用于不能量化的程度, 比如 いい　美味しい

而 けっこう 可以用于量化的程度 比如 長い　高い

```s
# 可以跟 なかなか 互换使用
結構美味しいですね

# 不可以跟 なかなか 互换使用
結構高いですね
```

<br>

### そう

### また
再见; 再, 又

<br>

黄桑, 我这边的沟通结果是这样的哈

1. 淘宝店会尽可能的给我派发做的不错的人, 但是不能成为固定人员, 因为淘宝有单子一般都是随机派发的
2. 淘宝这边会在设计接单的时候 提醒注意看指示

虽然不能完全的解决问题吧, 但是应该可以从某种程度上改善, 毕竟淘宝这种模式几乎都是套版的都形成肌肉记忆了 很难改

### とりあえず
暂且, 姑且, 先

表示在一串的事情中, 暂且先做这项

```s
# 暂且 先来3杯啤酒
とりあえず、ビールを3杯　お願いします

# 我们最开始 决定在北京 和 上海 将它卖出去
とりあえず、北京と上海で売り出したんですが
```

<br>

### それから
然后

在并列的事物后面添加一项事物的时候使用

本来动词て形在句型中是可以重复出现的 但是有的时候连接多了, 会使人觉得难以理解 而是会觉得花很啰嗦, 所以一般只会有て形连接两到三项 如果需要连接更多的动作 则可以把它们算开构成两个句子中间使用 それから 进行连接

```s
図書館へ行って、勉強します。それから、家へ帰って、手紙を書きます
```

<br>

### 十分

### もちろん

### ゆっくり

### まだ
还没有做, 尚未, 还没

**まだです:**  
但是这件事还是要做的, 只是现在没有做

<br>

**まだ + V(否定):**  
尚未, 还没

```s
三ヶ月パソコンを練習しましたが、まだ　あまり　できません
```

<br>

**技巧:**   
对 もう 提问的句子做否定回答的时候, 一般会使用 ``まだ　〜ていません`` 的表现形式
```s
小野さんはもう結婚していますか

まだ結婚してません
```

<br>

### すぐ

### ちゃんと
好好地, 的确, 完全

它表示这件事情本来应该是这样的, 应该这样去做, 那你有没有按照它应该的样子去做
```s
みなさん、机の上を　ちゃんと　片付けてくださいね

# 早饭本来就是应该规律的正经的吃, 你有没有好好吃饭呀
ちゃんと、朝ごはんを　食べてますか

# 妈妈带着孩子在街上碰到熟人, 孩子不爱打招呼
ほら、ちゃんと　あいさつしなさい
```

<br>

### ずいぶん

### ぜひ
非常, 务必, 一定

它通常用来表达自己的强烈的愿望 或者 对别人强烈的期待 请求 或者 提议, **通常它后面会接**

- 〜たい (表示自己的强烈愿望)
- 〜てください (对别人强烈的期待, 请求, 提议)
- 〜ましょう (对别人强烈的期待, 请求, 提议)

```s
今度　ぜひ　いえへ遊びにきてください
```

<br>

### そろそろ


## 常用接续词:

### でも
表示转折关系, 翻译: 但是 一般用于口语

<br>

### しかし
它是 でも 的书面表达, 可是, 但是, 单纯的转折关系

```s
物価が上がった
しかし、給料は上がらない
```

<br>

### ただし
不过, 只是, 但是;

后面接附加条件, 这个附加条件具有对**前半句内容的条件补充**

```s
このプールは誰でも使ってもいいです
ただし、有料です
```

有附加条件的转折用 ただし 只是单纯的转折使用 しかし

<br>

### そして
表示并列关系, 翻译: 而且 然后

<br>

### きちんと
好好地, 正经地

**きちんと、ちゃんと、しっかり的区别:**  
- きちんと: 很细致, 周全 (正面)
- ちゃんと: 整体上ok了就可以, 努力不要做到不好 (一点点负面)
- しっかり: 坚实, 可靠 (正面)

```s
運動会の準備が
  きちんと　できた (细节, 连细节都准备好)
  ちゃんと　できた (整体, 整体ok, 大部分)

部屋を
  きちんと　片付けなさい (仔细整理收拾, 该放哪放哪)
  ちゃんと　片付けなさい (不要乱糟糟, 整理好)

しっかりと結びました
しっかり勉強してください # 好好学 掌握知识要牢固
```

### なんだか
总觉得 / 不知道为什么

表示一种理由不明确的感觉或判断, 多用于表述说话者的心里活动

```s

```

<br>

### せっかく
好不容易, 特意

它的含义是对某种行为 或 事态的价值应充分利用

せっかく一般用来修饰动词, 但也有用在です前面的情况, 比如 せっかくですから 或者 せっかくですが

- 当别人劝说自己做某事的时候, 接受其好意的时候: せっかくですから
- 拒绝的时候: せっかくですが

```s
# 好不容易来北京, 不看看这是不能回去的
せっかく北京へきて、ここを見ないで帰ることはできませんよ

# 咖啡充好了请 / 谢谢 不过我没时间了, 先走了
コーヒー入れましたから、どうぞ
  せっかくですが、時間がないでので、失礼します
```
